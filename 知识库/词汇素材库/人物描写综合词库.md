| **感激** | 感激涕零 感恩戴德 感谢不尽 感同身受 |
| **高兴/欢喜** | 欢天喜地 大喜过望 兴高采烈 兴致勃勃 乐不可支 欣喜若狂 喜从天降 |

| **懊丧** | 灰心丧气 心灰意冷 万念俱灰 自暴自弃 黯然销魂 大失所望 |
| **忧愁** | 忧心忡忡 心事重重 忧心如焚 郁郁寡欢 闷闷不乐 愁眉不展 愁眉苦脸 |

### 心理活动例句（按情绪，可直接化用）
- **高兴**：我心里像喝了蜜一样，甜滋滋的 / 心里乐开了花。
- **紧张**：心一下子提到嗓子眼 / 怀里像揣了只小兔怦怦跳 / 像有十五只水桶打水——七上八下 / 像打鼓似的咚咚直跳。
- **难过**：心像被针扎一样 / 心疼得像刀绞，眼泪不住地流 / 像打翻了五味瓶 / 像掉进冰窖，从心顶凉到脚尖。
- **着急**：像热锅上的蚂蚁急得团团转。
- **害怕**：胆怯地低着头，不敢看那张阴云密布的脸 / 心里咯噔一下：这下准要坏事。
- **害羞**：脸刷地红到脖子根，如果地上有条缝真想钻进去。
- **内疚**：内疚、懊悔敲击着我的心，翻来覆去睡不着。
