## 整理原则（v2 精简）

- **frontmatter 极简**：仅 `tags`（不保留 source / ocr质量 / 整理状态 / 字数等溯源元数据）。
- **正文**：开头「一句话 + 写作价值/适用场景」TL;DR → 分级标题重建结构 → 能表格化尽量表格化 → 订正 OCR 错字（保持原意）→ 删版式污染 → 金句原文照录 → 末尾固定「🧠 整理思考」。
- **分层策略**：人设/公式/创作须知 深度整理；拆文中深度合集；词汇合并成专题词库；方法论合并同质 + 总览索引。
- **去重**：跨文件重复内容合并（如塑造法 OCR 版与 pymupdf 版、教程与单主题深度文、对白与 YY 桥段跨分类重复）。
- **跨文件互链**：用 `[[文件名]]` 标记关联，形成知识网络。
