## 一、言情 / 甜宠 / 霸总 / 虐 / 病娇（约 20 篇）

### 套路提炼
- **双马甲反差**：《他的VIP》男女主都有马甲（男主是大粉+影帝，女主也是大粉），小号暴露暗恋，搞笑误会贯穿。
- **「男德满分」甜点**：男主拒女演员借位、帮女主整理刘海——细节撒糖。
- **病娇/虐**：《不是月亮》病娇、《美人蛊》虐文，靠人设张力。
- **霸总反差**：《总裁秘书她不干了》女主决绝反撩。

### 案例清单
《他的VIP》（双马甲甜宠）、《待到繁华盛开时》《校草和姐姐》《恋与社死人》《不是月亮》（病娇）、《美人蛊》（虐）、《照殿红》（重生）、《总裁秘书她不干了》（霸总）、《枝枝计划》（脑洞言情）、《欢迎来到我身边》（电影剧本）；拆文《当男主他有钱又有病》《花开花落》《青梅倦》《青青子衿》《恨朝朝》《幼薇》《芙蓉乱》《玫瑰色的审判》；短篇《与顶流弟弟同居》《兰博基尼之恋》《泡他》。

> 🧠 甜宠核心 = 反差（双马甲/男德/霸总反撩）+ 搞笑误会。病娇/虐靠人设张力（见 [[病娇人设]]）。

## 二、系统 / 脑洞 / 弹幕 / 救赎（约 12 篇）

### 套路提炼
- **反差系统**：《绑定窝囊废系统后我成为龙傲天》——主角越窝囊越被豪门接纳（反套路）。**对比手法**：上一世努力背奢侈品 logo 被排斥，这一世窝囊发疯道歉反被接纳。
- **发疯文学/沙雕**：被嘲讽就「下跪磕头爆哭道歉」，沙雕反差。
- **弹幕/救赎**：《哥哥是反派也没关系》弹幕救赎。
- **系统流爽文**：靠系统金手指解决困难（主角不动手）。

### 案例清单
《绑定窝囊废系统后我成为龙傲天》（反差系统沙雕）、《春日来信》《争辉》（脑洞大女主）、《哥哥是反派也没关系》（弹幕救赎，2 篇）、《地师少女初出茅庐》（爽文）；拆文《儿媳被系统抹杀后》《妈妈攻略失败被系统抹杀》《全网嘲后绑定演技系统》《审判者实录》《苏梅梅的超市》《星火（和母亲穿越古代）》《整顿内娱计划》《交换人生》《置换人生》；《不再见》。

> 🧠 系统/脑洞核心 = **反差设定（窝囊废/弹幕/穿越）+ 沙雕或救赎基调**。系统是金手指，但爽点常来自「反套路」（越废越赢）。

## 三、复仇 / 大女主 / 逆袭 / 打脸（约 12 篇）

### 套路提炼
- **身份层层反转**：《青柳玉姿》女主身份「通房→扬州瘦马→青帮帮主妹妹→青帮帮主→顶替女二」，五层反转。
- **双视角揭秘**：女主视角 vs 男主番外视角，揭示「男主一直爱女主、女二将死是借口」的真相——伏笔回收震撼。
- **立意拔高**：《青柳玉姿》目标「废除奴隶制」，从个人复仇到群体解放。
- **伏笔**：妹妹的恋爱脑（实为保护姐姐而死）。
- **重生复仇**：《伥鬼朋友》恋爱脑闺蜜害死女主→重生「用魔法打败魔法」→闺蜜负债坐牢。**用情绪折线图标记读者上行/下行情绪**。

### 案例清单
《青柳玉姿》（大女主复仇，身份五层反转）、《重拳出击，未成年人渣》《青柳玉姿》；《公主心中无男人》（大女主，2 篇）、《沦陷》（言情大女主）、《争辉》（脑洞大女主）；《大女主复仇-换卷系统攻略失败》《打脸渣男后上北大》；拆文《穿成 po 文女主她妈》《主角光环下的非反派女配》《努力无敌》《只管努力》《哭着的学渣》《伥鬼朋友》。

> 🧠 复仇大女主核心 = **身份反转 + 双视角伏笔回收 + 立意拔高**。情绪靠「情绪折线图」精准控制（见 [[拆文方法论合集]]）。

## 四、追妻火葬场 / 银发（约 3 篇）

### 套路提炼
- **破镜重圆 + 罗生门**：《月光永不坠落》男女主双向误会（都以为对方在玩玩），靠**男主视角番外**揭示真相（他一直爱她、把她写进程序寻找她）。
- **多次反转**：未婚夫真假、还钱造谣、合约女配搅局。
- **17 万赞**的爆款，靠「破镜重圆大钩子」+ 反复拉扯。
- **银发/不婚**：《不婚醒悟》女主决绝离婚（见 [[短篇——追妻火葬场模板]] 的范本）。

### 案例清单
《月光永不坠落》（17 万赞，破镜重圆追妻）、《不婚醒悟》（追妻+银发拆文，也是 [[短篇写作公式——虐点写法]] 的案例原型）。

> 🧠 追妻火葬场核心公式见 [[短篇——追妻火葬场模板]]。爆款靠「破镜重圆大钩子 + 双视角揭秘真相 + 多次反转」。

## 五、悬疑 / 惊悚 / 诡异（约 15 篇）

### 套路提炼
- **诡异设定 + 反转**：瓶女/艳尸/颅针生子/骨粉罐——猎奇设定吸引点击，靠反转收尾。
- **不可靠叙述者/反转**：《我的爱人想杀我》《换角事件》《活在真空里》《人脸解锁失败》——日常下的惊悚反转。
- **丧尸/末世**：《日出桑榆》。
- **相亲/民俗诡异**：《聂小倩是相亲对象》《替嫁》《虫儿飞》。

### 案例清单
《日出桑榆》（丧尸）、《伥鬼朋友》；拆文《瓶女》《艳尸生子》《颅针生子》《骨粉罐》《审判者实录》；拆稿《洗衣机里的女儿》《诡异的准考证》；短篇《我的爱人想杀我》《换角事件》《活在真空里》《人脸解锁失败》《聂小倩是相亲对象》《替嫁》《虫儿飞》《星星以南》。

> 🧠 悬疑短篇核心 = **猎奇设定钩子 + 每 1000 字一反转 + 真相回收伏笔**（见 [[短篇写作公式——悬疑]]）。惊悚靠「日常下的异常」反差。

### 🧠 整理思考

**结构判断**：非世情题材拆文合集（约 75 篇案例，分 5 大题材 + misc）。我每题材抽样 1 篇代表作（甜宠 his VIP / 系统 窝囊废 / 复仇 青柳玉姿+伥鬼 / 追妻 月光永不坠落 / 悬疑 伥鬼朋友），提炼该题材套路，再列全部案例清单。单案细节不照搬。OCR 错字多（如 窝圳废→窝囊废、颠肌→腹肌、罗生国→罗生门），已订正关键处。

**知识提炼**（五题材核心套路）：
1. **言情甜宠** = 反差（双马甲/男德/霸总反撩）+ 搞笑误会。
2. **系统脑洞** = 反差设定（越废越赢）+ 沙雕/救赎基调，系统是反套路金手指。
3. **复仇大女主** = 身份层层反转 + 双视角伏笔回收 + 立意拔高，靠情绪折线图控情绪。
4. **追妻火葬场** = 破镜重圆大钩子 + 双视角揭秘真相 + 多次反转。
5. **悬疑惊悚** = 猎奇设定钩子 + 每 1000 字反转 + 真相回收伏笔 + 日常下的异常。

**贯穿五题材的共性**：**反转是短篇的生命线**（身份反转/真相反转/设定反转），且**双视角/番外**是揭示真相、制造震撼的常用高级手法。

**使用建议**：选定题材后，挑该题材 1-2 个案例做深度模仿（完整拆解见原始目录），套用对应套路。所有题材都重视「反转」和「情绪折线」。
