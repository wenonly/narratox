>
> **写作价值 / 适用场景**：这是**最重的角色塑造工具**。当你需要一个有厚度、有矛盾、有弧光的主角（尤其美强惨/复仇/救赎类），别急着写大纲，先用这份问卷把角色「采访」一遍——问完这八个维度，角色的内在逻辑、行为动机、戏剧看点会自动浮现。**沈烛这个范例本身就是一份合格的「美强惨人设模板」**，可直接套用改写。

> 1. **问卷框架**（每个「—」标题下的问题）：可复用的塑造工具，套到任何角色上。
> 2. **沈烛的作答**（每个问题下的引文）：美强惨复仇角色的完整范本，照着改名字/背景即可复用。

## 一、基本信息

**— 你喜欢自己的名字吗？它有什么特殊含义？**
> 名字？不过是个符号罢了。「烛」字意味着灰烬，就像我曾经的人生，被那场大火烧得一干二净。如今我重生，只为让仇人也尝尝化为灰烬的滋味。

**— 对你来说，什么最重要？**
> 复仇。那些害我家破人亡的人，必须付出代价。在他们彻底毁灭之前，我不会停下。

**— 兄弟姐妹？**
> 曾经有个妹妹，她才五岁，那么可爱……（眼神突然狰狞，拳头紧握）但他们连孩子都不放过，将她活活……（深呼吸，平复情绪）从那以后，我再没有亲人。

| 维度 | 沈烛的回答 |
|------|-----------|
| 民族 | 汉族 |
| 居住过的地方 | 曾住江南水乡的沈家大宅（最美好的回忆）；后四处漂泊，在黑暗的地下世界摸爬滚打；如今潜伏在仇人势力范围的阴影中 |
| 教育背景 | 幼时接受家族最好的教育，琴棋书画、兵法谋略无一不通；后为复仇自学各种暗杀、情报收集手段，在血与火中成长 |
| 旅行 | 去过最遥远的是极北之地，在冰天雪地中磨练意志，学习极寒环境下的战斗技巧 |
| 特长 | 精通各类兵器（尤其暗器和剑）；过目不忘，能迅速分析局势；精通易容术和语言伪装，可完美融入各种环境 |
| 职业 | 复仇者——为达目的，时而化作杀手，时而扮作…… |
| 工资 | 不需要。我需要的一切，都能用自己的手段获取，财富不过是复仇路上的工具 |
| 最喜欢的科目 | 曾喜欢诗词歌赋，向往文人雅趣；现在只对复仇相关的一切感兴趣（毒药配制、机关设计） |
| 特别的日子 | 沈家灭门那日（永远记得那冲天的火光）；手刃第一个仇人的那天（复仇的开始） |
| 父母 | 父亲是正直的商人，一生行善；母亲温柔贤淑，教我读书识字。他们都是善良的人，却落得那样的下场……（声音低沉，充满恨意） |
| 出生地 / 生日 | 江南沈家 / 早已不记得，也不再在意，只有仇人的忌日对我有意义 |
| 宠物 | 没有。我的身边只有冰冷的武器和随时可能背叛的「同伴」，温情对我来说太奢侈 |
| 交通工具 | 擅长骑马，也会驾驶马车，必要时徒步穿梭于街巷，隐匿行踪 |

**成长环境**：政治上处于各方势力明争暗斗的混乱年代，家族因涉权力斗争被陷害；经济上原本沈家富甲一方，灭门后一穷二白，只能靠自己在残酷世界求生。

## 二、人际关系

**— 谁指望着你？为什么？**
> 没有人指望我，也没有人值得我信任。曾经信任的人，要么背叛，要么早已不在。

**— 大家对你的评价？**
> 在暗处，他们叫我「修罗」，说我是从地狱爬出来的恶鬼；在明处，不同身份有不同评价——有人觉得我是神秘的富商，有人认为我是冷酷的杀手。

| 维度 | 沈烛的回答 |
|------|-----------|
| 大家最喜欢你什么？ | 哼，没有人喜欢我，也不需要。如果非要说，或许是我强大的实力能给一些人带来利益，但他们也畏惧这份力量 |
| 恋爱 | 感情？那是我早已抛弃的东西，它只会成为弱点。复仇的火焰会烧尽一切柔情 |
| 婚姻 | 从未想过，也不可能。我的生命里只有复仇，容不下其他 |
| 子女 | 不可能有，我不会让自己的孩子重蹈我的人生，在仇恨中痛苦挣扎 |
| 敌人 | 当年参与陷害沈家的所有人，上至权贵，下至爪牙，一个都不会放过 |
| 最崇拜谁？ | 没有崇拜的人。如果非要说，我只佩服那些能在绝境中求生的人，就像曾经的我 |
| 朋友 | 没有朋友，只有彼时的盟友。当利益不再一致，随时可以反目 |
| 和谁起争执 / 共度时光？ | 和所有阻碍我复仇的人起争执；和盟友在谋划复仇时共度时光。我不需要和任何人共度时光，除非是送仇人下地狱 |

> 🧠 关系网全是**负向连接**（敌人/盟友/背叛者），没有任何正向情感锚点。这是美强惨角色的典型配置——也是后续「救赎线」必须填的坑：**得有一个人/一件事能重新接住他的情感**，否则角色只有毁灭一个结局。

## 三、内心世界

| 维度 | 沈烛的回答 |
|------|-----------|
| 人生观 | 这个世界是弱肉强食的，善良和仁慈只会让人任人宰杀。只有让自己足够强大，才能掌控命运，讨回血债 |
| 你喜欢自己吗？ | 曾经或许喜欢过。但现在……我痛恨自己的弱小——如果当初我足够强大，家人就不会死。如今我只把自己当作复仇的工具 |
| 你的心魔是？ | 是灭门那晚的噩梦，是妹妹的哭喊，是自己无能为力的悔恨。这些日夜折磨着我，让我无法停下 |
| 乐观还是悲观？ | 我既不乐观也不悲观，我只相信自己的双手和手中的剑。复仇的道路注定是黑暗的，但我会一直走下去 |
| 真实的还是伪装的？ | 在这世上，真实会让人死得更快。我早已学会用不同的面具伪装自己，只有在独自一人时，才会面对那个被仇恨吞噬的自己 |
| 自信程度 | 对自己的实力有绝对自信，我为复仇而生，也将为此付出一切。但内心深处，始终有对失败的恐惧，害怕无法告慰家人 |
| 你有没有欺骗自己？ | 偶尔会想，如果一切没有发生，我现在会过着怎样的生活。但我很快就会清醒——那不过是无用的幻想，复仇才是我存在的意义 |
| 道德水准 | 为了复仇，我可以不择手段。那些所谓的道德，在我失去家人的那一刻，就已经崩塌 |
| 最想改变自己哪一点？ | 想改变自己的过去，回到那场大火之前，变得更强，保护好家人。但我知道这不可能，所以我只能改变现在，让仇人付出代价 |

> 🧠 **提炼**：内心世界的核心矛盾是「**强与弱的倒错**」——他痛恨的是「曾经弱小」（导致家人死亡），他追求的是「绝对强大」（复仇工具化）。这种「为复仇而自我物化」恰恰是他悲剧的根源：**他把复仇当成了存在的意义，而复仇一旦完成，他就失去了活下去的理由**（见 §七「白日梦」：复仇成功后想安静地死去）。这是美强惨角色最高级的戏剧陷阱。

| 部位 | 描写 |
|------|------|
| 体型 | 身材修长挺拔，肌肉线条流畅，充满力量感，常年训练让他身姿矫健 |
| 体态 | 行走时身姿优雅，却带着一股肃杀之气，仿佛随时能出手致人死地；站立时背脊笔直，不怒自威 |
| 头部 | 标准的俊美脸型，五官立体深邃 |
| 眼睛 | 一双修长的丹凤眼，眼尾微微上挑，眼神冰冷如冰，仿佛能看穿一切，偶尔闪过仇恨的火焰 |
| 鼻子 | 高挺笔直，为面容增添几分英气 |
| 嘴巴 | 薄唇紧抿，线条冷峻，很少露出笑容 |
| 皮肤 | 肤色偏白，却透着一股冷冽；脸上有几处淡色疤痕，是曾经战斗留下的印记 |
| 纹身/穿孔/伤疤 | 身上布满大小不一的伤疤，每一道都有一段痛苦的回忆 |
| 声音 | 低沉沙哑，富有磁性，语调平稳，很少有情绪波动，却能让人感受到其中的寒意 |
| 衣着 | 平时穿黑色劲装便于行动；伪装身份时按需变换——或华贵如富商，或朴素如平民 |
| 最先被注意到的特征 | 那双冰冷且充满仇恨的眼睛，以及身上若有若无的肃杀之气 |
| 健康状况 | 表面健康，实则因长期高强度训练和旧伤暗藏隐患，旧伤发作时会影响行动 |

> 🧠 外形描写的套路：**「美」靠五官细节（丹凤眼/高鼻/薄唇）+「强」靠体态与伤疤（肃杀/伤疤/矫健）+「惨」靠疤痕与旧伤隐患**。三要素各有对应的视觉符号，可拆开复用。

## 五、习惯 / 六、性格特征

**口头禅**：「血债，必须血偿。」
**标志性动作**：无意识地抚摸腰间的佩剑，或思考时习惯性地握紧拳头。

| 情境 | 沈烛的反应 |
|------|-----------|
| 开心时 | 没有开心的时候。复仇的火焰早已将情感焚烧殆尽，偶尔看到仇人倒下，会露出一抹冷笑 |
| 愤怒时 | 表面平静如水，眼神却更加冰冷，下手会更加狠辣，将敌人彻底毁灭 |
| 沮丧时 | 独自坐在黑暗中，擦拭武器，回忆过往，用仇恨重新点燃心中的斗志 |
| 伤心时 | 夜晚站在高处，望着家乡的方向，默默流泪，然后将泪水化作复仇的决心 |
| 害怕时 | 害怕复仇失败，会更加疯狂地训练，不断提升实力 |

**性格的坚强与软弱**：坚强在于无论遭遇多少挫折都不会放弃复仇信念；软弱在于面对家人的回忆时，内心会被痛苦淹没，变得脆弱。

**— 你强势的一面反而会成为弱点，为什么？**
> 过于强势的行事风格，容易引起他人的警惕和联合反抗，给自己带来更多阻碍。

**自律与自控**：极度自律，每天严格训练、规划复仇；自控力极强，能克制一切欲望，专注于复仇目标。

| 情绪来源 | 触发 |
|----------|------|
| 愤怒 | 看到仇人逍遥法外，或有人阻碍复仇计划，被愤怒吞噬，失去理智般展开杀戮 |
| 悲伤 | 想起家人的音容笑貌、曾经的美好，却再也回不去 |
| 恐惧 | 无法完成复仇、让家人亡魂无法安息；也害怕复仇后自己找不到活着的意义 |

> 🧠 **情绪反应表是个金矿**：把「开心/愤怒/沮丧/伤心/害怕」五态写成不同的行为模式，角色在剧情里就有了**可预测的反应库**。写新情节时，先查这张表——「这个情境他会怎么反应」就有答案了，不会写崩。

**— 如果只能做一件事，你会做什么？**
> 毫不犹豫地手刃所有仇人，让他们为自己的罪行付出代价，用他们的鲜血告慰家人。

## 八、写作角度（作者视角的元分析）

这份问卷最妙的是最后一节——它跳到**作者视角**，问的是「这个角色为什么好写/好看」：

| 问题 | 答案（角色设计逻辑） |
|------|---------------------|
| 我最喜欢这个角色哪一点？ | 他复杂而深刻的内心世界，以及在仇恨中挣扎却始终坚定的信念，充满了悲剧色彩和戏剧张力 |
| 读者为何与他快速建立联系？ | 悲惨的遭遇容易引发同情；强大的实力和坚定的复仇决心又会让读者产生敬畏和期待，想看到他如何逆袭 |
| 他的平凡与非凡之处？ | 平凡在于他也曾渴望幸福、有家人的温暖；非凡在于他能在绝境中崛起，将自己锻造成复仇的利刃，与命运抗争 |
| 他的处境看点 / 特殊在哪？ | 看点在于复仇过程中与各方势力的斗智斗勇，以及内心的矛盾与挣扎；特殊在于以一己之力对抗庞大敌人，明知希望渺茫却绝不放弃 |
| 核心需求 | 完成复仇让家人安息，同时在复仇过程中寻找自己存在的意义 |
| 标志性/高光时刻 | 在仇人面前摘下面具，看着对方惊慌的表情，所有隐忍和痛苦都值得；以及一场艰难战斗中以一敌众、扭转局势的瞬间 |
| 你属于哪类人？ | 被仇恨吞噬的复仇者，游走在光明与黑暗之间，为了复仇可以舍弃一切 |

**做过最糟糕的事**（道德灰度）：为获取情报，曾利用一个无辜女子的感情，在她失去利用价值后，眼睁睁看着她被敌人杀害，没有出手相救。
**最了不起的成就**：成功打入仇人的核心圈子，一步步瓦解他们的势力。
**最惊险的事**：一次伪装任务中差点被识破身份，最后只能靠杀人灭口解决，那次之后更加谨慎。
**秘密**：曾在妹妹坟前发愿，若不能复仇就自刎于坟前；身上有一处致命旧伤，每次发作痛不欲生，但不能让任何人知道。

### 🧠 整理思考

**结构判断**：这是「访谈型」的极致形态——它既是**塑造工具**（百问问卷框架），又是**完整范本**（沈烛的八维作答）。原文 OCR 错误极多（灰焊→灰烬、激味→滋味、峙痰→狰狞、摸肘滚打→摸爬滚打、翡观主义→悲观主义、自钾→自刎、眼睁罗→眼睁睁……），且有大量分栏重叠导致的重复行和「@发疯要拆书日万 幕布」等水印。我把散落的内容按原问卷的八个维度归位，去重、订正，并加了「🧠」批注点出每节的塑造要点。

**知识提炼**（这份问卷真正教给你的三件事）：
1. **复杂角色靠「问」出来，不靠「想」出来**。八个维度（基本/人际/内心/外形/习惯/性格/喜好/写作角度）覆盖了角色的全部面向，问完自然立体。比起凭空构思，这套问卷能逼出你没想到的矛盾点（如沈烛的「强与弱的倒错」「复仇完成即失去生存意义」）。
2. **矛盾是角色的发动机**。沈烛处处矛盾：强大 vs 怕失败、复仇工具 vs 渴望被理解、痛恨弱小 vs 因强大而被孤立。**每一个矛盾都是一条可展开的故事线**。
3. **「写作角度」一节是降维打击**——它让作者从「这个角色是谁」跳到「这个角色为什么好看」。这步很多人不会做，导致角色立体但无趣。

**使用建议**：
- **新立复杂主角时**，把这份问卷拷一份，逐题作答。别跳过任何一题——最不出彩的那题往往藏着最大的戏（如沈烛的「儿时白日梦」）。
- **卡文时**，回到「内心世界」和「习惯」两节，查角色的情绪反应表，让他按既定模式反应，剧情自然推进。
- **套用沈烛范本**：复仇/美强惨/救赎题材，可直接改名字、改背景、改复仇对象，复用整套人设骨架。
- **坑**：美强惨角色最易写成「苦情机器」——只剩恨、没有柔软面。务必给他留至少**一个柔软锚点**（沈烛的是妹妹的回忆），否则读者无法共情，只有猎奇。

**关联**：本篇是「塑造法型」的访谈变体，与人设档案中的「定义型」[[反差萌人设]] [[傲娇毒舌人设]] 互补——定义型告诉你「这类角色长什么样」，访谈型告诉你「怎么从零捏出一个有深度的角色」。可与「三步塑造高岭之花」（塑造法型）对照学习。
