> **一句话**：生活平淡无奇、缺乏追求、安于现状、懒散度日的角色。
>
> **写作价值 / 适用场景**：咸鱼是「反内卷」时代的情绪代言——在全员拼命的爽文里，一条理直气壮躺平的咸鱼自带**反差萌和代入感**。近年热门的「咸鱼女主」更是把消极人设玩出了新意（见 §六创新方向）。适合做轻松向/反套路文的女主，或给紧绷的主角群配一个解压的调节角色。

## 一、定义

生活平淡无奇、缺乏追求、安于现状，甚至表现出懒散特质的角色。

## 二、特征

### 心理状态
对未来没有明确规划，也没有强烈的改变欲望，持有一种**消极的宿命论观点**。缺乏自信，对自身能力不认可，觉得自己无法做出大成就，从而自我设限，不愿尝试新事物。

### 语言特点
常说消极、抱怨的话，口头禅如「算了吧」「就这样吧」「太累了，不想动」。交流中对他人积极想法反应冷淡，甚至泼冷水，传播负面情绪。

### 行为表现
| 维度 | 表现 |
|------|------|
| 行动迟缓拖沓 | 能拖则拖，对任务缺乏紧迫感 |
| 活动范围狭小 | 多局限于熟悉舒适的环境，长期宅家，极少社交或户外 |
| 缺乏毅力 | 做事常半途而废，遇困第一时间放弃而非克服 |
| 避重就轻 | 总选最轻松的事，面对困难优先考虑逃避（多一事不如少一事） |
| 舒适至上 | 以自身舒适感为行动导向，一旦舒适被打破就产生抵触 |
| 低欲望 | 对物质/精神追求都低，不图高品质生活、高额财富或事业成就，满足于基本生存 |

## 三、成长背景（咸鱼是怎么炼成的）

| 来源 | 逻辑 |
|------|------|
| **长期高压环境** | 经历长期高强度学习/工作压力（应试教育、高竞争岗位长期加班），身心疲惫，对努力奋斗产生抵触，以「咸鱼」姿态逃避 |
| **多次挫折打击** | 创业失败/感情破裂/考试失利等多次重大挫折未能走出，信心斗志被消磨，认为努力也无法改变现状 |
| **家庭环境影响** | 家庭过于安逸、父母过度溺爱包办，缺乏自主奋斗意识；或家庭氛围压抑、期望失衡，形成消极逃避性格 |

> 🧠 **提炼**：咸鱼几乎都有一段「曾经努力过却被压垮/溺坏」的前史。写咸鱼别只写躺平，要写他**为什么躺平**——这条前史是角色深度的来源，也是后续「被唤醒」剧情的钩子。

## 五、描写范例（原文照录）

> 他瘫在那张破旧的沙发上，眼睛直勾勾地盯着天花板，嘴里嘟囔着：「努力有啥用，还不是一样穷，不如躺着舒服。」阳光透过窗户洒在他身上，他却连挪动一下躲避刺眼光线的力气都没有，仿佛这世间的一切都与他无关，就这么心安理得地做着他的咸鱼。

> 在这个快节奏的社会里，他却选择了慢下来，或者说，是彻底停了下来。他辞去了工作，靠着微薄的积蓄生活，每天过着晒太阳、喝茶的悠闲日子。朋友们都不理解他，他却不以为然：「你们为了名利忙忙碌碌，我只想做条安静的咸鱼，享受生活的宁静。」

> 她每天早上总是在闹钟响了无数次后，才极不情愿地从被窝里钻出来，看着乱糟糟的房间，叹了口气，想着「反正也没人来，收拾了还会乱，算了」。洗漱完毕，便坐在电脑前，漫无目的地刷着网页，一天就这样浑浑噩噩地开始了，丝毫没有为自己的未来打算。

## 六、热门咸鱼人设创新方向（女主向）

单纯躺平已经不够看了，原文给出四种「反差创新」让咸鱼女主立住：

| 创新方向 | 玩法 |
|----------|------|
| **能力反差** | 表面懒散、对日常毫不上心，关键时刻却能凭隐藏的超强专业技能力挽狂澜 |
| **性格反差** | 日常慵懒，却有极度倔强的一面——在守护珍视之人/坚持内心正义时，能爆发出惊人能量 |
| **环境反差** | 把女主扔进竞争激烈的高压设定（如宫斗），众人都拼命争权，她却秉持咸鱼理念，反利用他人的倾轧开辟舒适小天地，无意间搅乱局势推动剧情 |
| **行为反差** | 看似无关紧要的咸鱼行为，却在庞大故事体系里引发连锁反应，改变女主人生 |

> 🧠 **提炼**：咸鱼女主的爆款公式 = **「躺平的壳 + 不躺的核」**。壳负责搞笑/代入/反套路，核负责关键时刻的高光反转。两条腿缺一条都立不住——只有壳是废柴，只有核是装咸鱼。

### 🧠 整理思考

**结构判断**：定义型人设。原文 OCR 把「心理/语言/行为/背景/案例/描写/创新」七层搅在一起，且夹大量分栏碎片（`二曙 0 1 24 oo Le | 一NI...`）。我按「定义→特征（心理/语言/行为表）→背景→案例→描写→创新方向」重排，特别把行为表现的六点收成一张表，把最有价值的「创新方向」单列成 §六。

**知识提炼**：
1. **咸鱼不是天生的，是被压垮/溺坏的**——前史是角色深度与剧情钩子。
2. **爆款咸鱼 = 躺平的壳 + 不躺的核**，靠反差制造高光。

**使用建议**：
- 咸鱼女主配宫斗/职场/修仙等高压题材，反差效果最强（环境反差）。
- 给功能配角（同事/室友/师弟）套咸鱼，是调节节奏、制造笑点的利器。
- **坑**：纯咸鱼没有「核」=真废柴，读者会烦；务必埋一个能爆发的隐藏面。

**关联**：与 [[小太阳人设]] 是镜像——一个消极避世，一个积极发光，做对照组很有戏。咸鱼的「能力反差」与 [[反差萌人设]] 的反差机制同源。
