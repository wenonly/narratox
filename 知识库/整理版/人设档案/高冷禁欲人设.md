> **一句话**：颜值出众，个性沉默内敛、极为高冷，对情感极为克制、清心寡欲——一旦陷入爱情，情感往往比常人更深沉浓烈。
>
> **写作价值 / 适用场景**：高冷禁欲是「禁欲系男主」的标配——越克制，破戒时的反差越致命，**「禁欲的外壳 + 深情的内核」**天然制造极致张力。适合霸总/师徒/仙侠的男主，是「越冷越苏」审美的核心人设。

## 一、定义

颜值出众，个性沉默内敛、极为高冷的角色。对情感极为克制，处于清心寡欲的状态；一旦陷入爱情，其情感往往比常人更深沉浓烈。

> 🧠 关键在最后一句——高冷禁欲的戏剧性全在**「克制 → 破戒」的反差**：平时越冷，动情时越烫。

## 二、特征

| 维度 | 表现 |
|------|------|
| **外表清冷** | 精致立体的五官，面部线条流畅却偏冷硬；眼神深邃犹如深潭，平静无波，让人难以窥探内心想法 |
| **气质高雅** | 举手投足带着优雅，任何场合都能凭自身气质脱颖而出，成为焦点 |
| **情绪内敛** | 不轻易表露情绪，始终保持沉稳冷静 |
| **着装讲究** | 穿裁剪合身的正装，每处细节都一丝不苟 |
| **保持社交距离** | 不喜欢亲密肢体接触和热情交流，言语简洁明了，给人难以亲近之感 |

## 三、行为准则

- **坚守原则**：内心有坚定的原则和底线，一旦认定的事，就坚定不移地执行。
- **克制欲望**：对物质和情感都有极强的克制力，注重精神层面的追求；感情上即便产生好感，也会极力克制。

> 🧠 **提炼**：「克制」是高冷禁欲的人设引擎——他克制得越狠，破戒时的冲击力越大。写他，前期的「冷与克」都是在为后期的「深情爆发」蓄力。

## 四、成长背景（冷是从哪来的）

| 来源 | 逻辑 |
|------|------|
| **名门世家** | 出生于名门望族，从小接受严格的礼仪规范和道德准则教育 |
| **童年缺爱** | 缺乏亲人的陪伴和关爱，只能独自面对生活，自我封闭，不善表达情感，对外界持防备态度 |
| **情感创伤** | 遭受过背叛、欺骗，用高冷禁欲的外表武装自己，将内心隐藏起来 |

> 🧠 **提炼**：高冷禁欲的「冷」多半是**创伤的盔甲**——要么被规矩框住（世家），要么被冷漠冻住（缺爱），要么被伤害逼出防备（创伤）。背景写好了，他的「冷」就有了让人心疼的根。

## 六、描写范例（原文照录）

> 他站在讲台上，身姿笔挺，气场强大。身上的白色衬衫一尘不染，领口的黑色领带系得整整齐齐。他的眼神扫视着台下的学生，目光如炬，犀利而冷静。高挺的鼻梁上架着一副金丝眼镜，更增添了几分清冷和禁欲的气息。他薄唇轻启，声音低沉而富有磁性，每一个字都清晰有力。

> 他走进会议室，众人的目光纷纷投向他。他身材高大魁梧，却又不失优雅。身上的深灰色西装剪裁合身，凸显出他宽阔的肩膀和修长的双腿。他的脸上没有任何表情，深邃的眼睛里透着一种让人捉摸不透的深邃。他径直走向会议桌的首位，坐下后，双手交叉放在胸前，静静地等待着会议的开始，整个过程没有说一句话，却已然掌控了全场的气氛。

> 她走在熙熙攘攘的大街上，人群中的她却显得格外孤寂。一头乌黑的长发随意地披在身后，几缕发丝被风吹到脸颊旁，更衬得她面容绝美却又透着疏离。她身着一件灰色的风衣，双手插在口袋里，眼神淡漠地看着前方，对周围的喧嚣充耳不闻，仿佛整个世界都与她无关。

> 她坐在花园的长椅上，周围繁花似锦，却都无法衬托出她的美丽。她的肌肤白皙如玉，双眸宛如一泓清泉，清澈却又冰冷。一头栗色的卷发随意地散落着，却有一种别样的慵懒和高雅。她身着一件淡蓝色的连衣裙，裙摆随风轻轻飘动，宛如一幅绝美的画卷。然而，她脸上那淡淡的冷漠，却让这幅画卷多了几分难以亲近的感觉。

> 🧠 描写规律：高冷禁欲的视觉符号 = **「冷色调（白衬衫/灰西装/淡蓝裙）+ 道具（金丝眼镜/领带）+ 拒绝感（双手交叉/插口袋/淡漠眼神）」**。核心技巧是写**「在场的疏离」**——人在人群中却「仿佛整个世界都与她无关」，冷感全在「与世界隔着一层」里。

### 🧠 整理思考

**结构判断**：定义型人设。原文 OCR 错字多（清袁欲→清心寡欲、绍潭→深潭、病探→窥探、高牙→高雅、波悟→魁梧、四嘻扩扩→熙熙攘攘、中离→疏离、喧器→喧嚣、白暂→白皙、球动→飘动……），描写段有重复，已订正去重。按「定义→特征表→准则→背景→案例→描写」重排。

**知识提炼**：
1. **戏剧性全在「克制 → 破戒」的反差**——越冷，动情越烫。
2. **「冷」多半是创伤的盔甲**——背景赋予冷以心疼的根。
3. **视觉符号：冷色调 + 拒绝感道具**——写「在场的疏离」。

**使用建议**：
- 霸总/师徒/仙侠禁欲系男主首选；配 [[软萌小白兔人设]] 搭「冷热/强弱」反差。
- 戏份分配：**前 80% 蓄力（冷与克），后 20% 爆发（破戒深情）**——破戒那一刻是全篇高潮，要给足。
- **坑**：高冷禁欲易写成「面瘫木头」。要给读者**「克制的缝隙」**——一个微表情、一次没忍住的关心，让读者窥见冰下的火，才有苏感。

**关联**：与 [[高岭之花人设]] [[阴冷孤僻人设]] 同属「冷调」谱系——高岭之花偏「高贵不可亵玩」，阴冷孤僻偏「创伤封闭」，高冷禁欲偏「克制深情」，三者冷的质地不同。与 [[霸道强势人设]] 都强势，但霸道强势外放控人，高冷禁欲内收克己，方向相反。
