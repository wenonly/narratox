## 一、基本信息

**— 她喜欢自己的名字吗？有什么含义？**
> 极为喜欢，常向众人解释「柔」字寓意温婉柔和，贴合她刻意塑造的娇弱形象。

**— 对她来说什么最重要？**
> 男主的偏爱和关注，以及由此带来的优越地位和物质享受——为此不惜破坏男主与女主的感情。

| 维度 | 林柔桑的回答 |
|------|-----------|
| 民族 / 出生地 / 生日 | 汉族 / 某小县城 / 3 月 14 日 |
| 居住过的地方 | 从小在小县城生活；后凭美貌和手段来到大城市，渴望扎根 |
| 教育背景 | 大专学历，文秘专业，学业平平，但擅长利用自身优势在社交中获取利益 |
| 旅行 | 喜欢去高档度假胜地，拍照发朋友圈展示奢华生活，塑造白富美形象，吸引男主注意 |
| 特长 | 擅长装无辜、扮柔弱，能精准捕捉男主情绪、适时展现脆弱；精通茶艺，善于用言语行为制造误会、离间他人关系 |

| 父母 | 父亲酗酒，母亲软弱，对她冷漠，只把她当补贴家用、为弟弟谋福利的工具 |
| 兄弟姐妹 | 家中长女，有个年幼的弟弟；父母重男轻女，她从小讨好父母获取资源，养成察言观色、工于心计的性格 |

> 🧠 注意背景：绿茶的「坏」有**原生家庭的根**——重男轻女、被当工具、极度渴望摆脱贫困。这条创伤线让她从「扁平的坏女人」变成「可理解的反派」，读者恨她也叹她。

## 二、人际关系

**— 谁指望着她？**
> 一些想通过她接近男主的人（希望借她与男主的关系获取资源）；她的父母也指望着她，把她当提款机。

**— 大家对她的评价？**
> 外人眼中她是温柔善良、楚楚可怜的「小白花」；女主和少数看清她真面目的人评价她「蛇蝎心肠、心机深沉」。

## 五、习惯 / 六、性格特征

**口头禅**（绿茶话术范本，原文照录）：
> 「都是我不好，要是我能更懂事一点就好了……」
> 「我真的不是故意的，你别生我的气好不好？」

**标志性动作**：双手绞着衣角，低头用眼角余光偷瞄男主；说话时轻抚男主手臂，表现出亲昵又无害的样子。

| 情境 | 林柔桑的反应 |
|------|-----------|
| 开心时 | 像小鸟一样依偎在男主身边，甜言蜜语不断 |
| 愤怒时 | 表面依旧保持微笑，但眼神会变得阴狠，之后会在背后报复让她生气的人 |
| 沮丧时 | 躲在房间里大哭一场，然后化好妆给男主发消息，装作若无其事地倾诉「烦恼」，寻求安慰 |
| 伤心时 | 眼眶迅速泛红，眼泪将要落下，声音哽咽地向男主诉说「委屈」 |
| 害怕时 | 立刻躲到男主身后，紧紧抓住男主衣角，颤抖着声音寻求保护 |

**最常用的面部表情及手势**：无辜的眼神、委屈的撇嘴、低头害羞的浅笑；常用手势有轻抚头发、摆弄衣角、双手合十放在胸前表示祈求。

**性格的坚强与软弱**：坚强在于为得到男主不择手段，能忍受各种委屈和指责；软弱在于害怕被男主彻底抛弃，一旦男主态度冷淡就慌乱无措。

**— 强势的一面反而会成为弱点，为什么？**
> 和女主争夺男主时，偶尔会表现出强势的占有欲，引起男主反感，让男主想起她的咄咄逼人，从而怀念女主的好。

**自律与自控**：在维持形象和吸引男主方面非常自律（严格控制饮食作息、时刻注意言行）；但嫉妒心作祟时缺乏自控力，容易做出冲动的事、暴露本性。

| 情绪来源 | 触发 |
|----------|------|
| 愤怒 | 女主和男主有接触，或男主对她态度冷漠，或有人揭穿她真面目 |
| 悲伤 | 被男主冷落、误解，或计划失败、无法得到男主全部的爱 |
| 恐惧 | 失去男主，被打回原形、重新过回贫困的生活 |

> 🧠 情绪反应表是反派的**「双面演技」说明书**——开心/沮丧/伤心/害怕全是演给男主看的（依偎/倾诉/哽咽/躲身后），唯独「愤怒」露真面目（眼神阴狠/背后报复）。写绿茶，要把「演给人看的反应」和「独处时的真面目」分开写。

### 🧠 整理思考

**结构判断**：访谈型（塑造工具 + 完整范本），与 [[给美强惨人物做访谈]] [[给咸鱼人设做采访]] 同构。原文 OCR 错字极多（林和桑→林柔桑、无豪→无辜、美狐→美貌、自插→自卑、仿禁→虚荣、最和从→最狼狈、含情防脉→含情脉脉、自梁→鼻梁、白哲细且→白皙细腻、无训→无害、轻对甜威→轻柔甜美、魂沂→魂魄、甜言密语→甜言蜜语、念归→念叨、委必→委屈、吊吊逼人→咄咄逼人、翡伤→悲伤、思春→愚蠢、奢修品→奢侈品、追麦火基场/追事火苦场→追妻火葬场、争寺→争斗、狼狐→狼狈……），且多处分栏重复、有整行乱码，已订正去重。按八维归位，表格化清单，保留口头禅与金句原文。

**知识提炼**：
1. **反派也要「采访」**——给绿茶一个匮乏的根（贫困原生家庭），她才从扁平恶毒变成立体可叹。
2. **「双面演技」是绿茶的写法核心**——演给人看的反应 vs 独处时的真面目，分开写。
3. **最高级的反派会「欺骗自己」**——林柔桑信了自己「渴望被爱的小女人」叙事，这层自欺让她可恨又可悲。
4. **她是「追妻火葬场」的发动机**——作为导火索推动男女主感情波折，功能明确。

**使用建议**：
- 都市/言情「绿茶第三者」「追妻火葬场导火索」，可直接套林柔桑模板。
- 这份范本示范了**如何把一个讨人厌的反派写出层次**——重点在「内心世界」的自欺和「写作角度」的情感博弈看点。
- **坑**：绿茶反派易写成「为坏而坏」的工具人。用这份问卷给她**创伤、渴望、自欺**，她才立得住；给她一个**旗鼓相当的女主对手**，戏才好看。
