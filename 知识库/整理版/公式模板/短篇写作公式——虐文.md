> **一句话**：开篇先给糖，再一刀刀夺走——主角越努力越失败，最后不得不面对残忍结局，靠「无力感」把读者虐哭。
>
> **写作价值 / 适用场景**：虐文短篇标准公式。适合写 BE 向、催泪向短篇，核心是用「先甜后苦」的反差和「行动代替描写」放大痛感。

## 一、人设要点

- **主角**：宁愿牺牲自己也要保护亲人或爱人。
- **亲人/爱人**：温柔乖巧却注定死亡（负债的单亲妈妈、重病不愈的白月光初恋）。

## 二、要点（虐的 4 个层次）

1. **直接制造冲突**：疾病、车祸、欠债。
2. **虐点分肉体与精神**：肉体是病痛折磨；精神是遭受丈夫背叛、只剩女儿相依为命。
3. **节奏**：开篇先展示幸福 → 冲突爆发（突发疾病、命不久矣）→ 主角**越努力越失败** → 最后面对残忍结局。
4. **用行动描写难过，不要直抒胸臆**：例——「她抱着骨灰盒坐了一天一夜」「她一遍遍拨打求助电话，却没有一个人接听」。
5. **BE 结局也给主角一点希望，这样更虐**：母亲去世了，欠债被母亲留下的钱还清了，可家里只剩她一个人。

## 三、公式骨架（起承转合 + 字数 + 情绪）

| 阶段 | 字数 | 节拍 | 情绪 |
|------|------|------|------|
| **起** | 0–1000 字 | ① 开篇铺垫情感，描写主角与爱人/亲人的幸福日常 | ↑ |
| | | ② 埋入伏笔暗示悲剧（咳嗽、流血、极端天气、不寻常举动） | ↓暗线 |
| | | ③ 爱人/亲人突发意外 | ↓ |
| **承** | 1000–4000 字 | ④ 主角为救人付出代价（卖房治病、弃学打工） | ↓ |
| | | ⑤ 主角和亲人互相隐瞒真相：亲人假装健康，主角假装还在上学 | ↓ |
| | | ⑥ 揭露双方真实情况——一个深夜吐血，一个为省钱吃馒头 | ↓ |
| **转** | 4000–8000 字 | ⑦ 冲突升级，为保护对方选择撒谎（母亲重病后告诉女儿自己要和别的男人远走高飞） | ↓ |
| | | ⑧ 主角发现母亲撒谎，陷入愧疚与绝望，爆发激烈争吵 | ↓ |
| | | ⑨ 亲人晕倒，死亡倒计时，医生宣判还剩一个月 | ↓ |
| | | ⑩ 两人和解，共度生命最后时刻，留下最美好的回忆 | ↑回光 |
| **合** | 8000–10000 字 | ⑪ 亲人在抢救室心跳停跳，主角悲痛欲绝 | ↓底 |
| | | ⑫ 整理衣物发现亲人留下的信，上面是密密麻麻的叮嘱和爱 | ↓余韵 |

## 四、情绪曲线

```
起(↑铺垫，↓暗伏笔) → 承(↓↓↓ 持续下坠) → 转(↓↓↓→↑回光返照) → 合(↓触底，余韵长鸣)
```

> 🧠 虐文的核心技巧是**「希望→夺走」的反复**：每次给一点光（和解/美好回忆），立刻再用更大的暗（心跳停跳）盖过。结尾那封信是「余韵刀」——人死了，爱还在，最虐。

---

### 🧠 整理思考

**结构判断**：公式型。原文 OCR 错字（幅牲→牺牲、命不久任→命不久矣、饥拨打→一遍遍拨打、还请→还清、揪入→埋入、愧妆→愧疚）已订正。按「人设 → 要点 → 起承转合表 → 情绪曲线」重排，12 个节拍收进表格。

**知识提炼**：
1. **虐 = 先甜后苦的反差 + 越努力越失败的无力感**。
2. **「行动描写难过」比直抒胸臆有效十倍**——抱着骨灰盒坐一天一夜，胜过一千句「她很伤心」。
3. **BE 也要留一点希望**，反差才虐得透；结尾用「余韵刀」（遗书/遗物）延长痛感。

**使用建议**：照起承转合表填，重点经营「互相隐瞒真相」和「回光返照」两段——这是虐文最催泪的引擎。换皮：把「母亲」换成任何注定失去的至亲/爱人都成立。
