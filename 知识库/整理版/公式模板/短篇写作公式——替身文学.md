> **一句话**：女主「有意识地」当替身——表面被男主当白月光的影子规训，实则利用替身身份达成自己的目的，后期反杀负心汉。
>
> **写作价值 / 适用场景**：替身文学短篇标准公式，且是**反套路版**（替身不是真卑微，而是下了一盘大棋）。适合写反转/反杀/大女主向短篇。

## 一、人设要点

| 角色 | 设定 |
|------|------|
| **女主（替身）** | 有意识地保留差异，只是为了用替身的身份达成自己的目的 |
| **白月光** | 看似完美温柔，实则嫉妒心爆棚 |

## 二、要点（反转是灵魂）

设置反转，让处于弱势的替身后期**反杀负心汉**——做替身只为了利用男主达成目的。反转的几种设定：
- 替身和白月光有交易
- 替身和白月光是姐妹
- 替身是男主竞争对手的女儿

> 🧠 反套路替身文的核心是**「替身的主体性」**——她不是被动受虐的影子，而是主动入局的棋手。前期所有的「卑微」都是演的，反转一刻全盘激活，爽感拉满。

## 三、公式骨架（起承转合 + 字数）

| 阶段 | 字数 | 节拍 |
|------|------|------|
| **起** | 0–1000 字 | ① 开篇揭露替身身份，男主用细节规训女主，要求她习惯要和白月光一模一样 |
| | | ② 交代替身协议（合约期限、替身义务、金钱交易） |
| **承** | 1000–4000 字 | ③ 男主经常盯着女主，叫出别人的名字 |
| | | ④ 女主趁男主不在，故意用白月光不喜欢的物品 |
| **转** | 4000–8000 字 | ⑤ 和白月光回国，与男女主偶遇，白月光羞辱女主 |
| | | ⑥ 白月光与男主复合，女主收拾行李离开 |
| | | ⑦ 女主觉醒，奔向自己的新生活 |
| | | ⑧ 男主不舍女主离开，找到女主，发现她笑起来和白月光一点都不像 |
| **合** | 8000–10000 字 | ⑨ 男主后悔，重新追求女主 |
| | | ⑩ 男主被女主嘲讽，男主才发现女主原来是有别的目的 |
| | | ⑪ 女主有了新生活，而男主失恋后每日买醉 |

## 四、结构内核

```
起(立替身身份+规训) → 承(暗藏反骨:故意用白月光不喜欢的) → 转(白月光回国+女主离开觉醒) → 合(男主反追→被嘲讽→揭目的,男主买醉收场)
```

> 🧠 替身反套路的爽点在**「合」段的双重反转**：先是男主后悔反追（读者以为要破镜重圆），再是女主嘲讽 + 揭露真实目的（原来她一直在利用他），最后男主买醉、女主新生。**从「被替身的受害者」翻盘成「下棋的操盘手」**，身份彻底逆转。

### 🧠 整理思考

**结构判断**：公式型，篇幅最短（513 字）但结构完整。原文 OCR 把「替身」全部误识成「蔡身」，还有「赵着」→「趁着」，已订正。11 个节拍收进表格。

**知识提炼**：
1. **反套路替身 = 替身的主体性**——她不是影子，是棋手。
2. **爽点在「合」段双重反转**——反追 → 嘲讽 → 揭目的，身份彻底逆转。
3. **反转要提前埋设定**（与白月光的交易/姐妹/竞争对手之女），逻辑自洽。

**使用建议**：开篇就把「替身协议」摆出来（合约/义务/金钱），让读者先信她是真卑微；转段让她「觉醒离开」；合段双重反转收尾。务必在前期埋好「真实目的」的伏笔，反转才不突兀。

**关联**：与 [[短篇——追妻火葬场模板]] 可叠加（替身文学 + 追妻火葬场是经典组合，见该篇搭配清单）。与 [[短篇写作公式——破镜重圆]] 都有「离开→反追」结构，但替身文多了一层「真实目的」的反转。白月光人设参考 [[白莲花人设]] [[绿茶人设]]（「看似完美温柔实则嫉妒」是其变体）。
