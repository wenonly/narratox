## 〇、先分清：叙述 vs 描写

| | 叙述 | 描写 |
|---|------|------|
| 做法 | 笼统交代、陈述事实 | 用形象语言把状态具体描绘出来 |
| 感觉 | 平淡如水 | 身临其境、形象感强 |

> 叙述：1945 年冬天，他在重庆街头死去。
> 描写：1945 年冬天，一具冰冷的尸体躺在重庆的街头，只见他双目睁着，浑身是伤，身上只穿着单薄的衣服，路上的行人匆匆走过，未曾多看他一眼。

**描写分类**：从角度分正面/侧面；从内容分**人物描写**（语言、动作、神态、心理、外貌）和**环境描写**（自然、社会）。

## 一、外貌描写

> 🧠 先泼冷水：**外貌没那么重要**——读者记得人物和情节，却记不清主角五官（除非某特征被反复强调）。所以别在五官上堆砌笔墨。

### 外貌写什么
容貌（五官面容）、身材（高矮胖瘦）、衣着（穿着打扮）、表情（喜怒哀乐）——以揭示思想性格、表达作者情感。

### 何时描写
1. **人物出场时**：让读者第一时间产生印象（如《一念永恒》《玄界之门》开头就带外貌）。
2. **特定情境下**：外貌与情境融合（心情不好皱眉、赴约刻意打扮）。但别动不动一出场就「长发飘飘」，会成累赘。
3. **想象/回忆中**：人物回忆某人时带其外貌。

### 三大方法

**① 抓典型特征，其余「藏」起来**
传统小说典范：关羽「髯长二尺，面如重枣」、张飞「豹头环眼」、孙悟空「尖嘴猴腮」、林黛玉「似蹙非蹙眉」、小龙女「肌肤胜雪，白衣轻纱」。**抓一个让人多看两眼、难以忘怀的点**（头发？眼睛？笑容？），其余略去。

**② 少用形容词堆砌，靠「细节」**
> 堆砌形容词 ≠ 建立形象。一个细节胜过千句「她很美」。
>
> 江南《九州缥缈录》：「一缕细细的淡金色头发从她雪白的帽兜中不老实地钻了出来，在面颊边淘气地卷起来，一颤一颤。」——头发从帽中钻出的细节，可爱跃然纸上。

**③ 注意描写顺序 + 抓主要特征**
- 从上到下、从外到内、从部分到整体（如「眉→眼→胸→整体气质」）。
- 或从整体到部分再回整体（「出尘如仙→紫衣/长发/长剑→美丽清雅」）。

> 🧠 铁律：**外貌描写必须符合人物设定的性格背景。**

## 二、心理描写（4 种方法）

心理描写直接写人物的七情六欲，让人物饱满立体、让读者共情。

| 方法 | 说明 | 例 |
|------|------|----|
| **① 直接描写** | 直接叙述交代内心活动，标志是心理动词（想/寻思/暗叹/嘀咕/盘算）。别全用「xx想」，用近义词避单调 | 「心想从穷乡僻壤来的破落少年，居然不被京都盛景所吸引……实在可笑」——《择天记》 |
| **② 内心独白** | 人物自思自语。分第一人称（《三生三世》「我思忖着……」）和第三人称（作者介入，《择天记》「陈长生很不理解今天发生的事……」）；内容上分自语（《赘婿》照镜确认换身体）和心里与人语（《赘婿》「唐明远你弄这个得花多少钱」） | 见左 |
| **③ 多角度呈现** | 用神态、外貌、动作、语言等细节**衬托**心理（不直写心理，让读者从外显推内心） | 《甄嬛传》华妃之死：知晓真相前刻薄语言+凌厉眼神+扑掐脖子=愤怒；知晓真相后又哭又笑+悲鸣+撞墙=绝望 |
| **④ 环境衬托** | 以景衬情（乐景写乐情、乐景写哀情），景物渲染烘托心理 | 《绝代双骄》「黑的牛，黄的马，白的羊，浩浩荡荡奔驰在蓝山绿草间……他的心胸已似突然开阔了许多」 |

> 🧠 **心理描写三原则**：① 写特定人物在特定情境必然产生的心理，不为描而描；② 别左一个右一个心理活动，只在关键情节/动作/表情时伴之以；③ 努力写细微感情波澜和复杂变化过程。**方法③「多角度呈现」最高级**——不直写心理，靠外显细节让读者自己推，最见功力（华妃之死是教科书）。

## 三、动作描写（4 要点）

动作描写通过个性化行动揭示性格，是人物描写里最重要的之一。

1. **写出连贯动作，准确运用动词**（《龙蛇演义》「深呼吸→走」「猛地抓住→扯→拖」连贯有力）。
2. **动作应符合人物身份性格和情境**。
3. **与神态、心理、语言巧妙联系**——动作不是孤立的，与情感/态度/姿势/表情联系时才有意义、才人格化。
4. **适当用比喻修辞**，更生动形象、增趣味。

> 🧠 例：「深呼吸、走」两个连贯动作 = 内心紧张但毅然决然；「猛地抓住、扯、拖」= 极度恨意。**动词的选择直接传递性格情绪。**

## 四、环境描写（六感 + 顺序 + 抓主景）

**自然环境描写从「六感」出发**，抓主要特点：

| 感官 | （雷雨天例） |
|------|--------------|
| 眼 | 窗外白茫茫一片，大地变汪洋，狂风暴雨肆虐，紫电撕裂虚空 |
| 耳 | 抽泣的风声、咆哮的雷鸣、哗啦啦的雨声交响 |
| 鼻 | 大雨将泥土腥味传递过来 |
| 口 | 寒冷天气让喝下的茶都有一丝苦意 |
| 身 | 感到一股沉沉的寒意 |
| 心 | 内心被扰得忐忑不安 |

**写景三要素**：
- **角度**：从六感出发（多数环境描写以「眼」和「心」为主）。
- **顺序**：远近结合、大小结合（如《寒门枭士》秋雨：远景原野 → 中景云/树林 → 近景地面泥泞）。
- **内容**：抓主要景物，别眉毛胡子一把抓。

> 🧠 神态描写（动态的肖像）：人在特定情境下五官的运动变化，一能表现内心、二可刻画性格。必须结合情境，不能为写神态而写。

### 🧠 整理思考

**结构判断**：描写技法总集。教程6（总论+动作+环境）、19（外貌）、21（心理）合起来覆盖描写的四大主块（外貌/心理/动作/环境），加上 [[对话描写技法]]（对话）和 [[打斗七公式]]（打斗），描写技法就齐全了。源文件干净。

**知识提炼**（四大块的核心一句话）：
1. **外貌**：没那么重要，抓一个典型特征「藏」其余，靠细节而非形容词，注意顺序。
2. **心理**：4 法（直接/独白/多角度/环境衬托），最高级是「多角度呈现」（不直写，靠外显推）。
3. **动作**：连贯动词 + 符合性格情境 + 联系神态心理 + 修辞。
4. **环境**：六感出发 + 远近大小顺序 + 抓主景。

**贯穿全文的共性**：**好描写 = 具体（非笼统）+ 有序（非杂乱）+ 服务人物（非为描而描）**。把叙述升级成描写的核心是「具体化」——给状态加上感官细节、动作连贯、典型特征。

**使用建议**：写流水账时，挑关键处把「叙述」改成「描写」（加六感、加细节、加连贯动作）。写外貌别堆形容词，找一个「让人多看两眼的点」。写心理优先用「多角度呈现」让读者自己推。
