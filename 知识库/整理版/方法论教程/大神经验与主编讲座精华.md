## 一、类型选择：没有最好的类别，只有最适合的（起点主编）

新人最爱问「流行什么、哪种容易红」。起点主编的答案：**没有最好的类别，只有最适合你的类别。**

- **冷门类型反而高订阅比例**：科幻签约少，但《小兵传奇》雄踞第一两三年，《超越轮回》《冒牌大英雄》单章订阅近两万——科幻是高订阅比例最高的类别。灵异曾被断言「必扑」，却出了《鬼吹灯》（全国畅销书榜前三、改编游戏电影漫画）。
- **热门 = 竞争激烈**：热门类型看的人多，但写的人更多，分类榜单难上、官方推荐少。冷门类型几乎每周有推荐，分类月票容易争（如武侠仙侠月票第六就有 107 票能领奖，同票数在都市只能排 23）。
- **根本在写好书**：订阅高低不取决于类型，取决于「书写得好、读者爱看」。怎么发挥作者全部水准？→ 选最适合自己（擅长/熟悉/喜欢）的类型。详见 [[网文入门与签约指南]] 选材三原则。

> 🧠 **冷门蓝海思维**：别盲目追热门。冷门类型竞争小、推荐易得，写出好书反而更容易出头。但前提是「适合你 + 写得好」。

## 二、心态：坚持 + 平常心 + 不跟风

**血红**（坚持论）：「现在网络创作环境不好，但既然投入了就努力坚持。当年遭受攻击我就是坚持下来的——如果没有面对攻击批评的心态和耐力，干脆别动笔。一开始写书不应针对物质利益，作为爱好坚持。」

**血红**（不跟风）：「尽量不要流俗，永远不要跟风，不要跟着别人走。跟风很难出优秀作品。尽量让自己的作品有特点。」

**血红**（平常心）：「不要成天钻营更新、稿费谈判之类的（交给编辑），认真码字就是，不要争这个争那个。写到一定功夫，自然而然就成了。」

> 🧠 三位一体心态：**坚持（扛住攻击）+ 平常心（不钻营）+ 不跟风（有特点）**。这是大神共识，也是新人最缺的。

## 三、创作习惯：大神们的「怪招」与共通点

| 大神 | 习惯 |
|------|------|
| **血红** | **不写大纲、不构思**，随手流出来的就是文字（「网上老大级别里唯一没大纲的」）；码字 6500-7500/时；人物原型采集自自身潜在性格（「写书的人多少都是神经病」）；**醉酒时写酣畅淋漓的打斗**（升龙道 magicparty 章节）；边写边 YY，结局自然出来 |
| **血红创作观** | 兴趣爱好驱动；读者意见考虑但不让其过多影响情节；很少写悲剧（符合大部分读者心思）；人物是复杂的综合体，知道自己喜欢和讨厌的东西来定人物好恶 |
| 失落叶（网游） | 兵器/Boss 命名：找关键词（命运/正义/勇气/力量/守护/天空）+「之/之刃」随意组合 |
| 月关 / 跳舞 / 酒徒等 | （详见原始访谈，多含构思/人物/情节的经验） |

> 🧠 **注意**：血红「不写大纲」是特例（天才型/灵感型），**新人别模仿**——绝大多数作者（含其他大神）还是要大纲（见 [[小说大纲写作总集]]）。可借鉴的是他的「平常心、不跟风、坚持、人物取自身边」。

## 五、读者与市场：大神怎么看

- **血红**：读者意见会考虑但不过多影响情节；市场说俗点就是钱，但他按兴趣写，很少写悲剧迎合。

### 🧠 整理思考

**结构判断**：大神经验/主编讲座聚类（约 25 篇访谈讲座）。这类文件性质特殊——多为口语化访谈，含大量生活八卦（如血红访谈一半在聊喝酒/找女友/不坐飞机），写作相关的精华散落其中。我只提炼写作相关精华（类型选择/心态/创作习惯/题材专精/读者市场），去掉八卦。源文件干净。

**知识提炼**：
1. **类型选择**：冷门蓝海（科幻/灵异高订阅比例），根本在写好书，选适合自己的。详见 [[网文入门与签约指南]]。
2. **心态三件套**：坚持（扛攻击）+ 平常心（不钻营）+ 不跟风（有特点）。
3. **创作习惯**：血红「无大纲灵感流」是特例，新人别模仿；可学「人物取自身边、平常心、不跟风」。大纲派见 [[小说大纲写作总集]]。
4. **题材专精**：各题材有专属心得（网游装备体系/官场权力博弈/悬念构架），见原始访谈或题材写法指南。

**使用建议**：基础不稳时先看系统教程（大纲/开篇/设定/爽点），有基础后再来读大神经验吸收实战智慧。心态卡壳（想放弃/被攻击/钻营）时回来读「心态三件套」。
