se。 字数: 1.1万

。 转身，我就出租房屋:

。 刚出狱的便宜五百，有精神病史的便宜六百，爱吵架的老太太便宜七百。

。 这下，轮到邻居跟我求侥了。
。 起
。 01邻居在我家门口对方杂物，沟通后被辱骂，我选择打消防电话 (困境冲突展示，重点刻
画反派形象，引起读者共鸣)
。 02消防员与反派发生冲突，消防员走后，反派盯上了我。 (展示危机，下钩子，引起期

。 承

冲突转变为武力冲突，暴打反派亚感+1)
。 06 "三疯'赶来救我，要反派给我道炊，反派不得不道歉。 (女主被救爽感+1，'三疯'细心体
贴为女主出头爽感+2，人设铺垫: 表面上是"三疯'实则是勇于痛击敌人的好人)
。 转
。 07花臂大哥要求反派搬家,三疯"痢来照顾我，老太太做饭，大哥和小弟护送我上下班。

。 08反派搬走了，我和"三疯"其乐融融，上网发现我被造h语，校长换下了我的课。 (冲突持
续升级，从武力冲突到网络暴力，影响到主角的事业)
。 09'三疯`蔡我生气，一起去找反派，反派还是不放弃造译，老太上去就是一巴掌。 (塑造
老太勇敢正义的形象，痛击反派爽感+1，暖心点+1)
。 合
。 10老太边骂边打，蔡我解释清楚了原委，周围路人感同身受把骂反派人渣，警笛声响。
(打反派爽感+1，周围人她骂反派爽感+2，结尾小钩子，警车是来抓谁的，引发期待)
。 11帽子叔朱来了，我给反派科普法律，反派下蹄求饶，我要求他给我录道歉视频，视频登
上热搜，反派被霸押。 (法律科普爽感+1，反派痛哭流涕更感+2，反派被加押爽感13，主
要写了主角送反派进局子的情节，前面几次打脸则是三疯轮番上阵，除掉反派是剧情的最
高潮内容，只有主角亲自上手，靠自己的实力除掉反派，才殉。)
。 12我在公开课上告诉孩子们，善恶并不绝对,三疯`看起来是恶，实则是善人，曾经我也
被认为是坏孩子，但那些坏只是别人认为的，并不是真的我，我很感恩上天补给我的亲
人。 (主题升华，见证主角的成长，增强故事的感染力，赋予故事特殊价值，引发读者的
思考)

。 @ 故事结构分析
。 开头300字如何切入: 以冲突开头，从第一视角描述邻居做的坏事，用画面感描写法(五感
描写) 增强代入感，迅速带起情绪。通过主角社恐人设，和反派鞭软怕硬人设，作对比，
使读者产生同情心;又通过物业了解反派，继续塑造反派人设，最后留下钧子，"主角打算
去找消防员"引起期待。
。 关键转折点分布: 1-3节被反派鞭辱，情绪下降，4-7节打脸反派，情绪上升，8节反派反

。 结局设计是否反套路: 反套路，主角与配角没有产生任何暖昧情居，而是像家人一样生活

。 4. 拆后总结

。 最值得学习的1个优点: 配角人设好，从三个看似恶名昭著的人，通过行为语言的展现，逐渐
发现角色的可爱之处，他们的动机是好的，伤害的是坏人，怎么能算坏呢，这个设计引人深
思，所以我们写作时设计角色的时也可以让他们变得复杂一些。

。 可用到下一篇的1个技巧: 小说第一节，作者借第三方之口塑造人物，这种方法可以让人物更
立体，单角度表述总会略显单薄和片面，通过别人口中出来的平价，视角更为客观，更让读者
信服。

。 5.素材收集

。 金句
。 试图玩弄舆论的人，终将被舆论反鸣。

。 在可以预见的将来，他和他的家庭，都会为他的行为付出代价。

。 无论发生多糟糕的事情，只要还有甜甜的西瓜吃、只要身边还有人能和你说说笑笑，一切
就不算太差劲。

。 善恶一念，在于动机，也在与行为。倘若我们用更加立体多面的视角去看竺他人，我们会
发现，恶不绝对，善也不绝对，它们在一念间发生变化。

。 这世界好像就是这样，每个人的身上都有无数标签，我们每时每刻都在给人贴标签，也被
人贴标签。可倘若我们愿意把标签据下来，用自己的眼睛认认真真去看。就会发现，原来
世界大不相同。

。 在我这里没有道德批判，我只在乎你是否受伤。
