言情脑洞电影《欢迎来到我身边》剧本分析
。 基本信息

。 类型: 奇幻+爱情+喜剧 (脑洞小耐文)

。 热梗: 失忆、破镜重圆、双向救赎

。 优点:

。 情节丰满不单调

。 奇幻与现实主义结合

。 基调轻松活泼

。 男女主人设新颖

。 悬念很足

。 人设:
。 男主: 大厂社畜，怪病患者，阳光开朗小太阳
。 女主: 美食up主，内心细肛，酒脱独立
。 三幕式结构
。 开始 (奇幻为主)

。 陈小舟因为一场意外有了幻视症状，只要看到黄色就会将人看成橡皮鸭子，这给他的日常

生活和工作带来诸多不便。
。 发展 (奇幻+喜剧)

。 命运在他遇到美食博主冯佳楠后发生转折。冯佳楠一旦出现，陈小舟的幻视就会消失，他
就能变成一个正常人。因此他对冯佳楠产生好奇，故意接近冯佳杭进行合作，在合作时俩
人逐渐产生好感。与此同时陈小舟的工作迎来了重大挑战，一旦失败面临失业。

。 结束 (言情为主)

。 冯佳楠从医生处得知，只有离开陈小舟让陈小舟独自克服幻视，才有机会恢复正常，冯佳
楠决定离开陈小舟。陈小舟为此伤心失落，偶然一天幻视真的消失了，这时候的陈小舟突
然恢复记忆，原来冯佳楠是他的前女友，大学时期的陈小舟帮助冯佳杭走出困境，又为她
策划帮助冯佳楠成为美食博主，俩人很是相爱却因为戒烟争吵分手，陈小舟为了戒烟去接
受电击治疗，电击故障使得他神经受损患了幻视的怪病。所以他的好友找到冯佳楠两人一
直在暗中帮助他对抗病魔，恢复记忆的陈小舟立刻去找到冯佳柄，重新在一起。

。 写作手法
。 倒序插叙

。 多次运用倒序插叙手法，来解决前面剧情留下的悬念。比如一开始男主做电击治疗结果造
成幻视，悬念就是为什么要去做电击，还有女主刚认识男主的时候，总是忧伤的偷偷看着
男主，到第三幕才解开悬念，原来女主假装不认识男主是想帮助他。剧情设置让每部分情
节都能悬念吸引着观众看下去。

。 前后呼应

。 前期剧情中的一些细节，到了高潮才一步步揭开，一些不经意间的细节和一些令人疑惑的
细节都得到合理的解释，让观众有种忧然大悟的感觉，比如男主失忆后做的独家泡面是恋
爱时女主教给他的，这些小细节反而更加动人。

。 奇幻设定

。 男主角电击失败产生的'幻视，让他看到所有人都变成鸭子，这一设定让这个爱情故事变得
更加丰满，也让平平无奇的情节更加有趣味性。奇幻设定在小说里也占据着重要的位置，
很多脑洞甜文都是以此为基础。比如常见的男主可以听见女主的心声，带来各种奇幻体
验,， 一步步了解女主然后相爱。还有常见的女主变成一只小动物后被男主收养，这类题材
的结构都颇为相似，都是从一个奇幻脑洞着手，因着与众不同和新奇视角而吸引读者看下
去，在后期揭开男女主的爱情故事。

。 人设新颖

。 男主是一枚社畜，而女主是一名美食博主。这些职业都是与当代社会的新兴产物，很富有
现代生活气息，情节都是日常琐事，很接地气，比如毕业季、电影院看电影、工作室拍
