描写五官跟形容外貌形容五官的成语


1.鼻如悬胆 : 形容人鼻梁挺直

2.樱桃小嘴 (口):形容女子的嘴唇小巧而红润，如同樱桃一般

3.唇红齿白 : 形容美人面貌姣好。亦作朱唇皓齿。

4.尖嘴猴腮 : 尖嘴巴瘦面颊。形容人长相极为丑陋。

5.长嘴獠牙 : 嘴大且牙齿外露。形容面貌凶恶丑陋。

6.眉横丹凤 : 形容女子美丽的眉毛。

7.鬓若堆鸦 : 形容两鬓乌黑的样子。

8.獐头鼠目 : 獐头小而尖，鼠目小而凸出。形容人相貌鄙陋，令人生厌。

9.龙眉凤眼 : 形容人英俊、气宇非凡。

10.浓眉大眼 : 形容人的眉目分明，带有英气。

11.眉清目秀 : 形容面貌清明俊秀，长相美丽。

12.螓首蛾眉 : 比喻女子的额头如螓首般，广而方正，眉毛如蛾的触须，长而纤细。

13.尨(ㄆㄤˊ)眉皓发 : 形容老人眉发尽白。

14.眉如墨画 : 眉色如墨画一般浓黑分明。

15.齿若编贝 : 形容牙齿如编排的海贝般洁白整齐。亦作齿如含贝、齿如齐贝。

16.蛾眉皓齿 : 细长眉毛和洁白牙齿。形容女子容貌明艳美丽。

17明眸皓齿 : 形容美人容貌明丽。(星眸皓齿)

18.明眸善睐 : 美人目光流转动人。

19.鹤发鸡皮 : 白发皱皮。形容老人的容貌。亦作鸡皮鹤发、鸡肤鹤发。

20.荑手纤纤 : 荑手纤纤形容女子的手柔细白净。

21.黑白分明 : 形容眼睛清澈明亮。

22.梨颊微涡 : 梨颊，梨花色的脸颊。涡，酒涡。梨颊微涡形容美女的笑靥迷人可爱。

23.炯炯有神 : 形容目光明亮而有精神。

24.画黛弯蛾 : 黛，妇女的眉毛。画黛弯蛾形容眉毛画的像蚕娥一样的弯细。

形容眼睛的词
【明眸】：明亮的眼睛。三国．魏．曹植．洛神赋：‘皓齿内鲜，明眸善睐。’
【星眸】：形容美丽的眼晴如星星般光亮。唐．章孝标．鹰诗：‘星眸未放瞥秋毫，频掣金铃试雪毛。’
【秋水】：其中一个意思是比喻如湖水般清澈美丽的眼睛。唐．白居易．筝诗：‘双眸剪秋水，十指剥春葱。’
【眼如秋水】：形容女子眼睛明亮美丽。老残游记．第九回：‘见那女子又换了一件淡绿印花布棉袄，青布大脚裤子，愈显得眉似春山，眼如秋水。’
【晶莹剔透】：光亮透明的样子。
【炯炯有光】：形容人眼睛明亮而有精神。
【炯炯有神】：形容目光明亮而有精神。清．薛福成．庸盦笔记．卷一．史料．江忠烈公殉难庐州：‘生平忠孝大节，出于天性，猿臂长身，目炯炯有神，顾盼磊然，与人交披肝沥胆，终始不渝。’
【目光如炬】：形容人怒视。南史．卷十五．檀道济传：‘道济见收，愤怒气盛，目光如炬，俄尔间引饮一斛。’形容目光有神。二刻拍案惊奇．卷五：‘中间坐着一位神道，面阔尺余，须髯满颏，目光如炬，肩臂摇动，像个活的一般。’
【目若朗星】：眼睛如星星一样的明亮。形容人的眼睛有神。三国演义．第五十三回：‘众视其人，面如重枣，目若朗星；乃义阳人魏延也。’
【目若悬珠】：形容目光明亮有神。汉书．卷六十五．东方朔传：‘臣朔年二十二，长九尺三寸，目若悬珠，齿若编贝。’
【顾盼生辉】：左右环顾，目光闪耀着动人的光彩。二刻拍案惊奇．卷二十二：‘一呼百诺，顾盼生辉。此送彼迎，尊荣莫并。’
【顾盼生姿】：左右环顾，目光炯炯动人。晋．干宝．搜神记．卷十八：‘华见其总角风流，洁白如玉；举动容止，顾盼生姿。雅重之。’
【双瞳翦水】：形容眼睛清澈明亮。明．周履靖．锦笺记．第九出：‘不要说什么，你只看他双瞳翦水迎人滟，风流万种谈笑间。’
