。 1人、场景+悬念切入
。 方法: 直接抛一个不同寻常的场景，让读者产生"为什么会这样"的疑问
。 案例: 我蹲在派出所走廊嘴包子时，警察把我哥的手机递过来。屏幕上是他最后一条朋友
圈，发在昨晚十点: 「这辈子做得最对的事，是娶了小雅」 。配图是他俩结婚照，媳子穿
着红裙子，笑起来露出两颗小虎牙。 但现在，我媳子的红裙子正泡在法医室的证物袋里，
而我哥躺在停尸间，手里携着半片带血的指甲。
。 解析: 用朋友圈照片与凶案场景的反差制造悬念
。 2、冲突+悬念切入

。 反常识事件或人物切入
。 方法: 用一个违背常理的设定打破预期，引发好奇
。 案例1:
违背常理的事件
。 我妈去世三年后，我收到她发来的快递。
。 快递单上的寄件人地址是 「市第一殖仪馆302室」 ，收件人是我的名字。
。 拆开盒子，里面是一包晒干的茉莉花，还有张纸条，闻迹是我妈生前的样子: "加因，
阳台的茉莉开了，记得泡茶喝。“
。 可我明明记得，她的骨灰是我亲手埋进墓园的，那天也是这样的晴天，风把纸钱吹得
漫天飞。
。 案例2:
违背常理的人设，《公主心中无男人》
。 我父皇从五年前就好像变了个人。人小时候他要求我食不言、滤不语，汤匀不能发出声
响，咀嚼不能张开嘴巴。
。 彼时我不小心打了个噶，涨红脸低着头等着他指正。他果然开呈了。
。 加的内容却是: 你能不能大口吃饭。
。 我余了，他给我夹了一只鸡腿，边嚼边说:“饭要大口吃才香，吃个饭搞那么紧张干
嘛? 消化不良晓得做?“
。 晓得，伐? 我半信半蜂地吃了一大口饭，他满意地国起眼睛。
。“对嗪，等会欧带你去看戏。 "他不爱听我喊父皇，总是自称欧驳。因为他喜欢被人叫
获。
。 我唯唯诺诺地点头: 是。每当佳节或志诞的时候，宫中就会唱戏。我自小看到大，其
实回回都看到打陵睡。
。 但父皇带我看的戏不一样。他给我看《况桥仙》，讲的是牛郎和七仙女的故事。看完
他问我: 你感受到了什么?
。 我思考着说: 感受到了他们坚贞的爱情?
。“错。 "他高深莫测地择了摇头,“你应该感受到牛郎看仙女洗澡并偷她衣服的操作太恶
心。 "我和身边的官女们都才出来一种忧然的表情。
