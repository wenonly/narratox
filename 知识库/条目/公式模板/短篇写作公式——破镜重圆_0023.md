短篇写作公式一一破镜重贺
。 要点
。 情感的极限拉扯: 酸涩感
。 宿命感: 不自觉的互相吸引，注定会在一起
。 不同时期的男女主人设变化(性格、阅历、社会身份、经济条件、被动方与主动方)
。 分手时
。 男主: 无动于囊，甚至放狠话，伤害女主
。 多年独自生活时
。 男主: 偷偷剧记女主，悔恨年轻的冲动行事
。 重逢后
。 男主: 纠结挣扎的内心，表面平静背后吃醋，曹谋已经制造和女主相遇
。 破镜怎么写: 误会重重，但不能有原则性问题，误会与人物性格自洽
。 被迫分开
。 被爱慕的第三者恶意拆散
。 被父母强制拆散
。 主动分开
。 性格骄傲，不届于解释，分开
。 恋爱摩擦后，和平分手
。 重圆怎么写: 避免水到渠成、平淡如水的和好，要铺垫好转折，情绪拉满
。 导语
。 突出角色的前后反差，设计反转
。 在一起时漠不关心
。 多年后后悔导求
。 写法结构案例
。 起: 开篇重估制造冲突
。 戏剧性: 女主听到，男主的现任和同事吐槽自己
。 修罗场: 女主假未婚夫和男主合约女友发生矛盾
。 重逢怎么拉扯: 尴众想逃又忍不住关注，视线频频相撞，吃醋却没有资格，明明很了解却嘴硬说不熟
。 承: 男女主不欢而散
。 承: 伤心回忆分手原因
。 铺垫误会
。 例: 男主冷漠，在一起时对女主漠不关心，女主家庭巨变后，分手出国，男主无动于囊。
。 转: 互相试探，误会加深
。 配角传递假信息
。 主角吵硬、扯谎不承认单身
。 合: 沉寂一段时间，放不下又不主动
。 起+承: 外力原因不得不见面 (实则是男主暗中促成的见面)
。 街角偶遇
。 公司共事、合作
。 邻居
。 同学聚会
。 感情升温: 女主嘴硬欣藏，男主满嘴嫌弃又上赶着找女主
。 转: 情感升温: 女主受伤，男主第一时间为她出头，发现女主是单身。
。 救助、帮助情节可多次出现，体现男主对女主的关心，铺垫一些感情甘谋已久的细节
。 合: 男主层求女主和自己在一起，女主犹豫
。 转: 男配女配离间男女主感情
。 目的: 加深误会，激化矛盾，拉扯读者情绪
。 (一般是假未婚夫、假女友、假婚约)
。 和主角没有实质性的关系，但因为种种原因要拆散男主女主
。 暗恋
。 仇恨
。 公司合作
。 制造由小到大的冲突等待男女主起矛盾
。 女配找女主放过男主
。 男主男配起争执打架
。 女主陷害女主，强制她离开这个城市
。 男配要丘女主订婚
。 合: 男女主闵矛盾，冷战
。 起: 感情升温，通过配角之口揭穿男主等女主很多年，从没有放弃过她
。 日记本
。 男主父母之口
。 社交账号
。 编入代码寻人
。 承: 说清误会，反思误会，甜密互动
。 原因一般是经济条件、家庭条件差距大或性格原因造成 (一个合理的产生误会的原因，逻辑自洽)
。 社交账号
。 编入代码寻人
。 承: 说清误会，反思误会，甜密互动
。 原因一般是经济条件、家庭条件差距大或性格原因造成 (一个合理的产生误会的原因，逻辑自洽)
。 转: 男主表明心意，误会解除
。 避免枯燥描写
。 反复追问后抛出答案
。 很久后旁观者揭晓
。 主角自己发现原由
。 合: 破镜重圆，和好大结局
。 搭配组合
