短篇追妻火葬场模板@友疯要拆书日万
。 短篇【追妻火基场】写作模板
。 一、核心
。 1. 极致的情绪拉扯: 短篇的核心是情绪，而不是复杂的剧情，读者来看的就是男主的'悔和女主的'痛"，以及
最后男主扎麦不得的“爽"，所有情节都要为放大这两种情绪服务
。 2. 快节奏、强冲突: 开篇即高潮，迅速进入“火葬场"阶段，不要慢热铺热，直接甩出最尖锐的冲突
。 3. 明确且巨大的“亏欠”: 男主对女主的伤害必须具体、深刻、难以原谅，例如: 替身文学、白月光陷害、流
产、当众羞辱、事业摧毁等，这是所有情节的基础
。 4. 女主的“决绝”: 这是爽感的来源，女主必须真正心死，不再流泪、不再回头、理性清醒、事业开挂
。 5. 男主的“疯批>追帮: 追麦手段要极端、疯狂、不顾一切，下跪、自残、破产、对抗全世界等，但要强调: 不
是卖惨就能被原谅，他的痛苦只是读者的亚点
。 6. 绝对的【HE]】或【BE】 :
。 HE: 不是简单的破镜重圆，而是女主基于自身利益 (如孩子、巨额补偿、事业助力) 选择接受一个“用起
来顺手”的男人，但爱已燃尽，关系的主导权完全在女主手中
。 BE: 男主彻底失败 (疯/死/破产) ，女主开启全新璀璨人生，身边有更优秀的追求者 〈(男二上位) ，形成
鲜明对比，爽感最大化
。二、结构
。 1000-2500字: 【引爆情绪】
。 开篇 (第1段) : 直接切入最高能场景!
。 模板示例:“顾总，签字吧，孩子我已经打掉了，这是离婚协议。”“陛下，臣过的这颗心，早在您为了林贵
妃将我们的孩儿打入冷宫时，就死了。”
。 快速倒叙/穿插回忆 (500字内) : 用最精炼的语言交代背景和亏欠。
。 她曾如何爱他: 省吃俭用供他读书、为他挡刀、在他低谷时不离不弃
。 他如何伤害她: 认错白月光、让她捐肾/输血、在小三面前让她下跑、害她失去孩子或重要亲人
。 决裂名场面 (1000字) : 女主抛出决定性证据或话语，彻底击碎男主傲慢，完成身份转变
。 女主: 冷静、麻木、言语如刀， "我不是在跟你商量，是在通知你。”顾先生，我们很熟吗? ”
。 男主: 从震惊、不层到最初的慌乱。"苏晚，你闹够了没有? ”“.……你说什么? 孩子? 什么孩子? 1 ”
。 【第一阶段爽点】: 女主潇洒离开，男主第一次体验到失去的恐慌
。 2500-5000字: 【制造对比】
。 女主线 (事业/颜值/桃花开挂) :
。 凭借过去隐藏的才华 设计、医术、投资眼光) 或贵人 (男二) 相助，迅速嘱起
。 变美、变富、变强大，身边出现优质温柔的男二/追求者
。 此处可写1-2个打脸原圈子里看不起她的人的爽快小情节
。 内心0S: 专注于搞钱搞事业，偶尔想起过去只觉得当时自己眼瞎
。 男主线 (追妻火基场启动) :
。 发现真相: 通过偶然听到的对话、发现的日记、白月光自爆等，知晓当年所有误会和女主的付出，痛不欲
生
。 疯狂寻找: 动用一切力量却找不到，或找到后发现自己已无法轻易接近她
。 初期追妻: 送花、送房、送公司被女主当面拒绝或转手送人/捐掉，男主开始体验“痛”。
。 5000-8000字: 【高潮】
。 男主升级追妻手段:
。 卖惨: 大雨中下跪一夜、为她挡酒瓶头破血流、故意让自己生病
。“帮"她: 为她扫清事业障碍 (但可能被她嫌弃多事) 、疯狂打击男二 (适得其反)
。 终极杀招: 放弃重要项目/皇位、散尽家财、用自残的方式证明爱意 (“只要你回来，我把命都给你")
。 女主的应对: 始终冷静。
。 对卖惨:“顾总，苦肉计对我没用，请叫你秘书来给你收记。”
。 对帮助:“谢谢，这是多少钱?我按市场价三倍付给你，两清。”
。 对自残: “拨打120，这里有人需要和急救。”
。 【核心殉点】: 男主付出一切，却发现自己所有的努力在女主眼中都是笑话和负担，他彻底明白，他永远地失
去了她，这种绝望感是火薄场的最高潮

”三、总结
。 1. 开篇一定要炸: 前三段必须抓住读者，直接扔出离婚`、“打胎`、*决裂'等重磅炸弹
。 2. 多用对比: 过去的女主VS现在的女主; 过去的男主VS现在的男主;，男主此时的痛苦VS他曾经的冷漠
。 3. 心理描写是重点: 尤其是男主的心理，要细腻刻画他的后悔、恐慌、绝望，女主的心理则要突出她的清醒和
。 4. 对话要犀利: 女主的对话要刀刀见血，男主的对话要从高傲到阜微，多写'打脸"对话
e。 5. 避坑:
。 不要让女主轻易心动: 男主做一点好事女主就动扬，是大鼠
。 不要洗白男主: 他的坏要坏得具体，后悔要悔得深刻，但不必强行洗白
。 男二工具人: 男二可以优秀，但不要抢了男主“火葬场"的戏份，他是用来让男主醋疯和衬托女主魅力的
。 节奏勿拖理: 一万字没追麦，还在虐女主，读者就跑光了
