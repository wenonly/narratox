> **一句话**：用「百问角色访谈」八维逼问法，从零捏出一个有血有肉的咸鱼角色——本文以宫斗咸鱼「阮软」的完整作答做示范。
>
> **写作价值 / 适用场景**：这是《给美强惨人物做访谈》的姊妹篇，同一套问卷工具，换了个**反差极大的示范角色**——把咸鱼塞进修罗场般的后宫。阮软这份作答本身就是「咸鱼 + 宫斗」题材的现成人设模板，可直接套用。

**用法**：① 每节标题下的问题是可复用的塑造工具；② 引文是阮软的完整范本作答，照着改背景即可复用。

## 一、基本信息

**— 对名字的态度与含义？**
> 觉得名字软糯可爱，和自己性格很搭。父亲取这个名字，是希望她一生温柔顺遂，没料到她入了后宫，反倒把「躺平」贯彻到底。

**— 生命中最重要的事？**
> 吃好喝好睡好、收集宫里好吃的点心方子、偶尔逗逗御花园的小猫。

| 维度 | 阮软的回答 |
|------|-----------|
| 民族 / 出生地 / 生日 | 汉族 / 江南苏州 / 七月初七 |
| 居住过的地方 | 自幼在江南水乡长大；后选秀入宫，一直住在偏僻的长乐宫 |
| 教育背景 | 学过些女红、诗词，大多没坚持下来，只对厨艺相关的书感兴趣 |
| 旅行 | 入宫前跟兄长游历江南；入宫后最远只到御花园最深处，连冷宫都懒得去 |
| 特长 | 品鉴美食一流（精准说出糕点配料）；睡觉功夫了得（嘈杂环境秒入睡）；擅长装傻充愣，巧妙避开后宫纷争 |
| 职业 | 后宫小嫔妃，位份常在，多年未晋升 |
| 工资 | 每月固定例银，位份低数额不多，但哥哥们偷偷送钱送物，日子倒也滋润 |
| 最喜欢的科目 | 食谱、糕点制作类书籍 |
| 特别的日子 | 生辰能吃到兄长送的江南美食；每月发月例可去小厨房加餐的日子 |
| 父母 | 父亲是江南商人，精明能干；母亲是大家闺秀，温婉贤淑，教她闺阁礼仪 |
| 兄弟姐妹 | 家中排行最小，上面三个哥哥，对她宠爱有加，常往宫里送新奇玩意儿 |
| 宠物 | 御花园捡的橘猫「团团」，胖嘟嘟的，整日和她一起睡懒觉 |
| 交通工具 | 基本靠走；偶尔乘宫人抬的软轿，但嫌麻烦，多数时候懒得坐 |

**成长环境**：政治上朝堂党争激烈，后宫各方势力明争暗斗；经济上家中经商，衣食无忧，入宫后家族为保平安小心谨慎。

## 二、人际关系

**— 谁指望着你？**
> 长乐宫的小宫婢们，指望她别惹事、好安稳度日；兄长们希望她在宫里平平安安，别卷入是非。

**— 大家对你的评价？**
> 后宫众人笑她胸无大志，是「扶不起的阿斗」；皇帝觉得她单纯无害，没有后宫女人的弯弯绕绕；太后认为她没什么威胁。

| 维度 | 阮软的回答 |
|------|-----------|
| 大家最喜欢你什么？ | 没心机，相处轻松；总能搜罗到宫里好吃的，愿意分享 |
| 恋爱 | 对皇帝没什么男女之情，只当是顶头上司，偶尔被翻牌子，就当换个地方睡觉 |
| 婚姻 | 选秀入宫，成为皇帝的嫔妃，婚姻于她不过是换个地方睡觉 |
| 子女 | 无，也不想卷入生儿育女的纷争 |
| 敌人 | 无——她对任何人都没有威胁，没人把她当对手 |
| 最崇拜的人 | 御膳房的李厨，觉得他厨艺出神入化，梦想是吃遍他做的所有美食 |
| 朋友 | 同为低位嫔妃的柳常在（常聚在一起分享点心）；御花园花匠王伯（给她送新鲜花酿） |
| 起争执 / 共度时光？ | 几乎没有争执，除非有人抢她点心；常和柳常在、小宫婢们、团团共度；希望能陪她一起吃喝睡大觉的人 |

关系网全是**低冲突、高日常**的温暖连接（宫婢/柳常在/花匠/猫），没有敌人——这正是咸鱼在后宫活下来的策略。也是这类文「无反派也能好看」的底气：靠日常糖和反差撑戏。

## 三、内心世界

| 维度 | 阮软的回答 |
|------|-----------|
| 人生观 | 人生苦短，及时行乐。在这深宫里，活着舒坦最重要，不争不抢，安度余生 |
| 是否喜欢自己？ | 挺喜欢自己的生活状态，觉得比那些勾心斗角的嫔妃快乐多了 |
| 心魔 | 害怕兄长们因自己在宫里受牵连；担心以后吃不到好吃的 |
| 乐观还是悲观？ | 绝对的乐观主义，总能在糟糕的事情里找到乐子 |
| 真实还是伪装？ | 完全真实，懒得伪装，在后宫做最自在的自己 |
| 自信程度 | 对吃、睡等「特长」相当自信，对其他事毫不在意 |
| 有没有欺骗自己？ | 从不欺骗自己，想要什么就去争取（为吃点心会厚着脸皮求御膳房） |
| 道德水准 | 坚守基本的善良，不会害人，见到受欺负的宫婢也会帮忙 |
| 最想改变自己哪一点？ | 希望能勤快些，多去宫里逛逛，别总待在长乐宫 |
| 做过最糟糕的事 | 有次太饿，把柳常在藏的最后一块桂花糕偷吃了，惹她气了半天 |
| 最了不起的成就 | 成功在宫里生存多年，没被卷入任何纷争，还胖了十斤 |
| 最尴尬的事 | 有次在皇帝面前打盹，还打呼噜，醒来后皇帝笑了好久 |
| 最在意的事 | 美食、睡眠、家人的安危 |
| 秘密 | 在长乐宫的地窖里藏了一堆点心，心情不好就去「寻宝」 |
| 如何描述自己？ | 「我就是后宫一条咸鱼，没啥大志向，吃饱睡好就成。」 |
| 信仰 | 信奉「美食即正义」，觉得没有什么烦恼是一顿美食解决不了的 |

阮软的内核是**「通透」**——她的咸鱼不是糊涂，是看透了后宫的本质后主动选择的活法。「最了不起的成就：没被卷入纷争，还胖了十斤」这句，把咸鱼哲学写出了通透的喜感，是全篇最妙的一笔。

## 四、外形特征

| 部位 | 描写 |
|------|------|
| 体型 | 娇小可爱，鹅蛋脸 |
| 体态 | 走路慢悠悠，总懒洋洋的；坐姿随意，偶尔跷二郎腿 |
| 头部 | 圆润 |
| 眼睛 | 大大的杏眼，笑起来弯成月牙，透着机灵劲儿 |
| 鼻子 | 小巧可爱，鼻头肉肉的 |
| 嘴巴 | 樱桃小嘴，唇色粉嫩，时常沾着点心碎屑 |
| 皮肤 | 白皙细腻，透着健康光泽（大概是吃好睡好的缘故） |
| 耳饰 | 耳朵上两个小耳洞，戴着兄长送的珍珠耳坠；无纹身伤疤 |
| 声音 | 软糯清脆，语调慢悠悠，带江南特有的软糯口音 |
| 衣着 | 常穿淡粉色、鹅黄色等柔和颜色的宫装，款式简单；不爱华丽首饰，只在腰间挂个香囊，里面装着桂花 |
| 最先被注意到的特征 | 肉嘟嘟的脸蛋、总挂着笑意的眼睛，以及身上若有若无的点心香气 |
| 健康 | 身体倍儿棒，吃嘛嘛香，偶尔因贪吃积食 |

外形三件套：**「软（鹅蛋脸/肉鼻头/软糯口音）+ 懒（慢悠悠/跷二郎腿）+ 馋（点心碎屑/桂花香囊/点心香气）」**。尤其「身上若有若无的点心香气」——把馋写进体香，是咸鱼外形最妙的辨识符。

## 五、习惯 / 六、性格特征

**口头禅**：「好困啊」「好饿啊」。
**标志性动作**：揉眼睛、打哈欠；无聊时转着手中的帕子。

| 情境 | 阮软的反应 |
|------|-----------|
| 开心时 | 眼睛亮晶晶的，像小仓鼠一样抱着点心大快朵颐，还会哼小曲 |
| 愤怒时 | 鼓起腮帮子，气呼呼地跺脚，但很少真生气，除非有人动她的食物 |
| 沮丧时 | 趴在床上，抱着团团，一边抱怨一边吃点心，吃完就又开心了 |
| 伤心时 | 躲在被窝里哭一会儿，然后起来吃顿好的，就觉得没那么难过了 |
| 害怕时 | 浑身发抖，往人多的地方躲，嘴里念叨着「怎么办」 |

**性格的坚强与软弱**：坚强在于勾心斗角的后宫能坚守本心、不为名利所动；软弱在于胆小怕事，遇麻烦能躲就躲，不敢正面冲突。

**— 强势的一面反而会成为弱点，为什么？**
> 偶尔为了食物「据理力争」，可能会得罪人，但她很快就会忘记，不放在心上。

**自律与自控**：极度不自律，想吃就吃、想睡就睡，毫无时间观念。

| 情绪来源 | 触发 |
|----------|------|
| 愤怒 | 有人抢她食物、欺负她宫婢、破坏她睡觉氛围 |
| 悲伤 | 收到家中不好的消息、看到可怜的小动物、被人误解 |
| 恐惧 | 打雷、老鼠、被皇帝厌弃赶出宫（倒不是舍不得皇帝，而是怕没了这安稳日子） |

## 七、喜好

| 维度 | 阮软的回答 |
|------|-----------|
| 天赋 | 对美食有超乎常人的敏感度，能尝出食材好坏和配料比例；模仿能力强，看一遍就能学会简单的点心制作 |
| 兴趣爱好 | 吃点心、睡懒觉、逗猫、研究新的糕点方子 |
| 政治立场 | 不参与任何党派纷争，保持中立，只想安安静静过小日子 |
| 收藏 | 收集宫里各种点心方子、好看的帕子、小巧的食盒 |
| 最喜欢的食物饮品 | 桂花糕、玫瑰酥、杏仁茶 |
| 最喜欢的音乐 | 江南小调，轻柔舒缓，适合伴着入睡 |
| 最喜欢的书 | 《御膳房秘传糕点谱》《各地美食大全》 |
| 最喜欢的运动 | 散步（仅限于从长乐宫到御膳房的距离） |
| 最喜欢的颜色 | 淡粉色、浅黄色 |
| 儿时白日梦 | 开一家属于自己的点心铺子 |
| 现在白日梦 | 在宫里有吃不完的点心，不用遵守那么多规矩 |
| 喜欢的礼物 | 新出的点心、精致的食盒、好看的糕点模具 |

**— 如果只能做一件事？**
> 吃遍天下所有美食，然后找个山清水秀的地方，开家点心铺子，安度余生。

## 八、写作角度（作者视角的元分析）

| 问题 | 答案（角色设计逻辑） |
|------|---------------------|
| 我最喜欢这个角色哪一点？ | 她的通透与豁达，在残酷的后宫中，依然保持对生活的热爱和纯真 |
| 读者为何与她快速建立联系？ | 她的「咸鱼」心态和对简单快乐的追求，能让读者在快节奏生活中找到共鸣、心生向往 |
| 她的平凡与非凡之处？ | 平凡在于渴望安稳简单的生活；非凡在于能在复杂的后宫中不被同化，活出自我 |
| 她的处境看点 / 特殊在哪？ | 身处波谲云诡的后宫，却能独善其身，用「躺平」的方式化解危机，这种反差极具戏剧性 |
| 核心需求 | 在后宫中平安度日，满足口腹之欲，偶尔能和家人相聚 |
| 标志性 / 高光时刻 | 宫宴上不顾形象地大快朵颐，惹得皇帝和太后哈哈大笑，化解了一场潜在纷争；凭美食天赋改良御膳点心，受众人喜爱 |
