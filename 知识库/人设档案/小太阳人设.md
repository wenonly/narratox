> **一句话**：充满正能量、乐观开朗、温暖他人的角色，像小太阳一样照亮周围。
>
> **写作价值 / 适用场景**：小太阳是治愈系和甜宠文的「情绪发动机」——一个永远发光的角色能瞬间拉满作品的暖度，也最适合做「救赎向」故事里把阴郁主角焐热的那个人。适合甜宠/校园/治愈题材的女主或暖男配。

## 一、定义

充满正能量、乐观开朗、温暖他人的角色。

## 二、特征

| 特征 | 表现 |
|------|------|
| 性格阳光开朗 | 总是保持积极乐观的心态 |
| 热情友善 | 对人热情大方，善于结交朋友 |
| 善良有爱心 | 心地善良，富有同情心，关心他人的需要和感受 |
| 充满活力 | 精力充沛、活力无限，仿佛永远不会疲倦 |
| 具有感染力 | 积极向上的态度能感染周围人，让人感受到生活的美好 |
| 真诚待人 | 真诚坦率，不虚伪不做作，言行一致 |
| 乐于助人 | 主动关心他人需要，见人遇困毫不犹豫伸出援手 |

## 三、行为准则

- **积极面对生活**：无论遇到什么困难和挫折，都能保持积极乐观的心态。
- **传递正能量**：通过言行举止，向周围人传递积极向上的能量和价值观。
- **尊重他人**：尊重他人的意见和选择，不强行把自己的观点强加于人。

## 四、成长背景

| 来源 | 作用 |
|------|------|
| 幸福的家庭环境 | 温馨和睦的家庭，父母给予充分的关爱和支持 |
| 良好的教育背景 | 接受良好教育，培养了正确的价值观和人生观 |
| 丰富的人生经历 | 经历过挫折困难，但没有失去信心勇气，反而更坚强成熟 |

**提炼**：小太阳的「暖」要有底——要么被爱滋养（幸福家庭），要么从苦里淬出甜（经历挫折仍乐观）。第二种（笑过伤的小太阳）比第一种（天生傻白甜）更有厚度，也更戳人。

## 六、描写范例（原文照录）

**外貌（原文有重复录入，去重后）：**

> 她有一张圆圆的脸蛋，像软软的团子，满满的胶原蛋白，白皙的肌肤透着淡淡的红晕，笑起来脸颊上会浮现出两个浅浅的酒窝，可爱至极。眼睛又大又圆，清澈明亮，犹如黑宝石般闪烁着纯真无邪的光芒，睫毛浓密卷翘，像是两把小扇子，每当她惊讶或者开心的时候，眼睛就会睁得更大，显得格外灵动。她的眉毛弯弯的，像是月牙儿，给人一种温柔甜美的感觉。鼻子小巧而挺拔，嘴唇总是微微上扬，粉粉嫩嫩的，看起来就像一颗熟透的樱桃，让人忍不住想要咬一口。
>
> 经常扎着一个高高的丸子头，丸子蓬松饱满，发丝柔顺光滑，偶尔会有几缕碎发垂落在脸颊两侧，为她增添了几分俏皮和可爱。有时候也会梳着马尾辫，马尾随着她的走动而左右摇摆，充满了青春活力。

**行动（发光瞬间）：**

> 她弯下身子，耐心地教一个小女孩跳舞，脸上始终挂着温柔的笑容。阳光洒在她身上，她就像一个发光体，将温暖传递给每一个孩子。她会为孩子们的每一次进步鼓掌欢呼，笑声回荡在花园中，吸引着更多孩子围拢过来，她的存在，让这个花园充满了生机，宛如一个被小太阳眷顾的乐园。

> 她声音清脆，带着十二分的热忱，眼睛亮晶晶的，瞬间点燃了整个办公室的氛围。她一边眉飞色舞地讲解，一边手舞足蹈地比划，那股子活力，让原本沉闷的职场都焕发出别样生机，仿佛有她在，再棘手的难题都能迎刃而解。

描写规律：小太阳的刻画 = **「暖色外貌（圆/红晕/酒窝/亮眼睛）+ 发光行动（主动靠近/感染他人）」**。写她时多用「阳光洒在她身上」「像发光体」这类光感比喻——这是小太阳区别于普通活泼角色的视觉符号。
