> **一句话**：外表文雅、举止得体，内心却道德败坏、行为不端的角色。
>
> **写作价值 / 适用场景**：斯文败类是「优雅与邪恶」的极致缝合——西装革履的皮囊下藏着算计，这种**表里反差**天然制造惊悚与张力，是悬疑/犯罪/商战题材最迷人的一类反派。适合做幕后黑手、伪善上位者、或主角「最信任的人却最危险」的设定。

## 一、定义

外表文雅、举止得体，但内心却道德败坏、行为不端的角色。

## 二、特征

| 特征 | 表现 |
|------|------|
| **外表斯文** | 穿着得体，言谈举止优雅，常戴眼镜，给人高雅、有教养的印象 |
| **内心阴暗** | 表面温和谦逊，实则心狠手辣，擅长利用虚伪和欺骗达到目的 |
| **双重人格** | 外表与内心形成强烈反差，表面绅士，背后却可能做出伤天害理的事 |
| **脆弱与狂躁并存** | 外表冷静，内心深处却可能充满孤独、失落或不安全感 |
| **隐秘行事** | 不直接与人冲突，而是通过挖苦、暗示、诋毁等方式达到目的 |

## 三、行为准则

- **利用他人**：擅长操纵他人情绪，以达到自己的利益。
- **伪装高尚**：表面上追求道德和正义，实则内心充满自私和恶意。

> 🧠 **提炼**：斯文败类的「斯文」是**武器**，不是修养——他用教养降低你的防备，用体面掩护下手。写这个角色，功夫全在「不动声色」四个字。

## 四、成长背景

| 来源 | 逻辑 |
|------|------|
| 文化衰落 | 社会环境变化导致传统美德淡漠 |
| 个人选择 | 缺乏对道德规范的认同，追求个人利益而忽视社会责任 |
| 家庭影响 | 成长中受到不良家庭教育或环境影响 |

## 六、描写范例（原文照录）

> 他戴着金丝眼镜，一副温文尔雅的模样，却在暗地里策划着一场阴谋。

> 他总是用微笑掩饰内心的阴暗，仿佛一切尽在掌握。

> 宴会上，他穿梭在人群中，手中端着一杯红酒，偶尔浅酌一口。笔挺的西装勾勒出他修长的身形，每一个动作都尽显绅士风度。与旁人交谈时，他微微颔首，认真倾听，适时地给出回应，脸上始终挂着温和的笑容。然而，当他的目光扫过远处的目标时，那眼底瞬间闪过的寒意，却被他巧妙地隐藏在斯文的表象之下。

> 他站在办公室窗前，背对着门，双手抱胸。透过玻璃，能看到他笔挺的西装和挺直的脊背。听到门被推开的声音，他没有回头，只是淡淡地说：「坐吧。」声音平静无波，却莫名带着一种让人无法抗拒的威严。待来人坐下，他才缓缓转身，脸上带着一丝若有若无的笑意，可那笑意却没有到达眼底，深邃的双眸中透着让人捉摸不透的光芒。

> 🧠 描写规律：斯文败类的点睛之笔永远是**「一个泄底的微表情」**——眼底闪过的寒意、笑意没到达眼底、目光扫过目标的瞬间。全程斯文是平庸反派；斯文 + 一瞬破绽，才是高级斯文败类。

### 🧠 整理思考

**结构判断**：定义型人设，篇幅短但结构清晰。原文 OCR 较干净，仅结尾描写段重复了一次。我按「定义→特征表→准则→背景→案例→描写」重排，去重。

**知识提炼**：
1. **「斯文」是武器不是修养**——他用教养掩护下手。
2. **高级感来自「一瞬破绽」**——全程完美的反派无趣，完美的皮 + 偶尔泄的底，才让人毛骨悚然。

**使用建议**：
- 悬疑/犯罪：做幕后黑手或「最信任的人最危险」的设定，效果最佳。
- 商战/权谋：伪善上位者，靠斯文败类质感提升反派魅力。
- **坑**：别让他过早露底——斯文败类的戏份全靠「读者比角色晚一步发现真相」的悬念撑着。

**关联**：与 [[白切黑人设]] [[腹黑心机人设]] 高度同源（都是「表善里恶」），区别在于斯文败类更偏「社会精英/教养伪装」，白切黑更偏「人畜无害伪装」。与 [[反差萌人设]] 方向相反——反差萌是「表硬里软」（讨喜），斯文败类是「表善里恶」（危险），别写反。
