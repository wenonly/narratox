> **一句话**：以搞笑、无脑欢乐为特点，靠荒诞的情节和行为制造幽默，行事与常人不同，充满反差与荒谬感。
>
> **写作价值 / 适用场景**：沙雕是「快乐源泉」型角色——在严肃/紧绷的剧情里，一个毫无包袱、满嘴热梗的沙雕能瞬间破冰、调节节奏，也最适合做搞笑向/沙雕文的主角。适合沙雕/轻松/吐槽向题材，或给任何故事配一个解压担当。

## 一、定义

以搞笑、无脑欢乐为特点的角色，通常通过荒诞的情节和行为来制造幽默效果，行为方式往往与正常人不同，充满反差和荒谬感。

## 二、特征

| 特征 | 表现 |
|------|------|
| 思维跳脱 | 逻辑与众不同，思考问题的角度令人意想不到 |
| 情绪转换极快 | 上一秒兴高采烈，下一秒可能因一点小事变得沮丧 |
| 热梗频出 | 使用当下热门网络用语、流行段子，巧妙融入日常交流，说话充满趣味 |
| 气氛扭转大师 | 无论多沉闷、紧张或尴尬的场景，都能凭独特言行迅速打破僵局，让气氛变轻松 |
| 喜欢展现自我 | 毫不掩饰沙雕特质，不怕成为焦点，以自信夸张的方式展现搞笑言行 |

## 三、行为准则

- **梗王式互动**：善于利用网络热梗与他人互动，根据不同对象和场景，巧妙抛梗来活跃交流氛围。
- **欢乐至上原则**：一切言行的出发点都是为了制造欢乐，无论日常交流还是面对困难，都以幽默诙谐的方式应对。

## 四、成长背景

| 来源 | 逻辑 |
|------|------|
| 家庭宽松 | 父母对其行为管束较少，给予充分自由，角色得以发展个性、不受束缚，形成无拘无束的性格 |
| 氛围乐观 | 成长中一直处于乐观积极的氛围，身边人多心态阳光，善于用幽默化解困难 |
| 网络文化熏陶 | 成长于网络文化盛行的时代，从小接触各种网络段子、搞笑视频，能熟练运用网络热梗，形成独特思维模式 |

## 六、描写范例（原文照录，沙雕台词是核心资产，一字不改）

**对恐怖怪物的骚话输出（封不觉）：**

> 他不但不害怕，反而眼睛放光，兴奋地说：「哇，你这造型太酷了，简直就是恐怖界的时尚先锋！你是不是兼职做模特啊，这身材比例，这面部线条，不去走 T 台可惜了！」怪物的他说得一愣，还没反应过来，他又接着说：「你要是能和我合作，我保证让你成为全副本最红的怪物，到时候粉丝肯定多得数都数不过来，说不定还能出周边，比如你的玩偶，肯定大卖！」怪物听了，气得哇哇大叫，而他却在一旁笑得前仰后合。

**大碴子味魔法腔（艾莉丝）：**

> 此时，她正对着身旁同样一脸懵的同学，操着一口大碴子味的魔法腔大喊：「哎呀妈呀，这魔法阵可太炫了，我感觉我要是往里头一钻，指定能开启『盲盒』，抽出个超级魔法秘籍啥的，到时候我就『拿捏』了整个魔法世界，成为魔法界的『显眼包』，谁见了我不得喊一声『666』！」说着，她还真就跃跃欲试地往前冲，差点一头栽进魔法阵，吓得周围同学赶紧把她拉住。她却一脸不在乎，拍了拍身上不存在的灰尘，继续念叨：「怕啥呀，这不是没进去嘛，要是进去了，我指定能上演一出『魔法版甄嬛传』，把那些魔法大佬都『团灭』！」

**上课吐槽数学（李闹闹）：**

> 李闹闹顶着一头刺猬般的绿色头发，那发色绿得扎眼，活像一颗会行走的西兰花。他穿着一件印有「我不是懒，我是在充电」字样的卫衣，卫衣的帽子上还别着几个卡通徽章，徽章上的图案都是些搞怪的表情包。上课铃响了，老师在讲台上讲着复杂的数学公式，李闹闹却在下面偷偷和同桌嘀咕：「兄弟，你说这数学公式，是不是像加密文件一样，得有个『解码神器』才能看懂？我感觉我脑子就是个『2G 网』，根本加载不出这些公式的正确答案，要不咱也搞个『外挂』，比如偷偷在桌子上贴个小抄，来个绝地求生式作弊，说不定能『逆风翻盘』，从学渣逆袭成学霸，走上人生巅峰！」同桌被他逗得差点笑出声，赶紧捂住嘴巴。他却越说越起劲：「要是真能逆袭，我就把这张桌子供起来，每天上香，感谢它是我的『上分神器』，然后再发个抖音，标题就叫『学渣的逆袭之路，全靠这张神奇桌子』，指定能爆火，涨粉百万不是梦！」

描写规律：沙雕角色的灵魂是**「用错位的语域对待正经场景」**——对怪物谈经纪约、用东北话解魔法、把数学当加密文件。**正经设定 + 不正经反应 = 沙雕**。梗要用得密、用得自然，像角色真的长在网络里。
