> **一句话**：表面上温柔可人，内心阴暗险恶的女性反派——清纯是皮，算计是骨。
>
> **写作价值 / 适用场景**：绿茶是「情感操控大师」——靠示弱、撒娇、装受害者激发他人的保护欲，暗中挑拨离间、攫取利益，是言情/都市/宫斗里最招人恨（也因此最有效）的反派女配。适合做「第三者」「白月光替身文里的原配」「女主路上的绊脚石」。

## 一、定义

表面上温柔可人、内心阴暗险恶的女性反派角色。

## 二、特征

| 特征 | 表现 |
|------|------|
| 外表清纯无邪 | 甜美的外貌和温柔的气质，给人无害、单纯的印象，让人容易产生保护欲 |
| 善于伪装 | 隐藏真实意图，常表现得单纯天真 |
| 情感操控 | 通过示弱、撒娇、哭泣等方式激发他人的保护欲，达到自己的目的 |
| 人前人后两面派 | 人前乖巧善良懂事，人后阴险狡诈，靠造谣、挑拨离间达到目的 |
| 善于利用暧昧 | 刻意维持朦胧暧昧的关系，让男子觉得自己在她心里是特别的，又不明确表达，让对方陷入纠结 |
| 喜欢装受害者 | 产生矛盾时，一副委屈巴巴的模样，让人觉得错都在对方，自己是无辜的受害者 |
| 以自我为中心 | 一切行为以满足自己的利益和需求为出发点，不顾及他人 |
| 不择手段 | 为达目的不惜欺骗、利用、伤害他人 |

## 三、行为准则

- **善于利用他人**：察觉他人的弱点和需求，利用这些来控制和操纵他人，让他人为自己所用。
- **注重形象塑造**：非常在意自己在他人眼中的形象，重视穿着打扮、言行举止。
- **以退为进**：竞争失败后保持纯洁形象，而在成功后对对手发起挑衅。

**提炼**：绿茶的核心战术是**「示弱即进攻」**——她的每一滴泪、每一次撒娇，都是精准投放的情感武器，瞄准的都是他人的保护欲与愧疚感。写绿茶，要写她**「装」的清醒**——人前含泪，人后冷笑。

## 四、成长背景

- **缺乏安全感**：成长中缺乏父母关爱，内心缺爱，通过绿茶行为获取关注和认可。
- **家庭环境复杂**：父母离异、家庭暴力等，使其学会通过伪装保护自己。
- **自卑与虚荣心驱动**。

## 六、描写范例（原文照录）

> 她一双水汪汪的大眼睛，无邪地看着他，仿佛受了天大的委屈。她轻轻地咬着嘴唇，眼泪在眼眶里打转，让人看了心疼不已。

> 「姐姐，我听闻你近日身子抱恙，特来探望。」她声音轻柔，仿若春日微风拂过耳畔，带着恰到好处的关切。说罢，微微颔首，眼中满是真挚，似是真的为对方忧心忡忡。可那低垂眼眸中，转瞬即逝的一抹得意，却被有心人尽收眼底。

> 与人交谈时，她总是微微咬着下唇，眼眸中噙着泪意，泫然欲泣：「都是我不好，若我能早些想到办法，也不会让公子这般为难。」这般示弱模样，任谁瞧了，都只道是个善良柔弱、满心为他人着想的姑娘，又有谁能想到，这看似无害的表象下，藏着怎样的心思。

> 她惯爱穿素净衣裳，今日一袭月白绫罗裙，裙摆缀着细碎雏菊，随着她莲步轻移，似有淡淡花香飘散。那宽大衣袖半掩着如削葱般的玉手，偶尔露出一截手腕，肌肤胜雪，叫人不敢直视。发间仅斜插一支碧玉簪，几缕碎发垂在脸颊两侧，更添几分楚楚动人之态。

描写规律：绿茶的招牌是**「人前端着泪，人后露得意」**的瞬时切换——「转瞬即逝的一抹得意，却被有心人尽收眼底」是写绿茶最经典的一笔：让读者（或女主）看到她藏起来的真面目，而剧中其他角色还被蒙在鼓里，戏剧张力就出来了。
