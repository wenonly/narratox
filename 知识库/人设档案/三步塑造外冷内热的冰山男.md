> **一句话**：用「三问搭框架 → 细节填落地 → 五阶段递进」三步，塑造一个外表冰冷、内心滚烫的冰山男——让读者在慢慢发现他的温柔中，比直白描写更上头。
>
> **写作价值 / 适用场景**：与 [[三步塑造高岭之花]] 同系列的实操工序，专攻「外冷内热」型男主。适合写傲娇/嘴硬心软/默默守护型男主的感情线，是「甜宠/暗恋/救赎」向的核心男主写法。

> 🧠 本文是「外冷内热冰山男」的塑造工序，结构与方法与 [[三步塑造高岭之花]] 完全一致，可对照学习——高岭之花偏「高贵不可亵玩」，冰山男偏「冷漠有理由、内里有温度」。

## 第一步：三个问题搭框架

### Q1. 他为什么冷？（给冷漠找合理理由，让读者共情）
| 来源 | 举例 |
|------|------|
| 创伤 | 童年被抛弃、父母离世、寄人篱下 |
| 环境 | 长期高压环境 |
| 性格 | 天生情感淡漠 |

### Q2. 他对谁热？（明确「热源」输出对象，完善人设）
- **互补型女主**：活泼开朗，像小太阳一样融化他（如《偷偷藏不住》段嘉许）。
- **救赎型女主**：与他有类似创伤，两人互相治愈。

### Q3. 他如何伪装？（冷是表，热要藏）
- **行为**：以「顺手顺路」为借口帮助女主。
- **语言**：用反问或嘲讽掩盖关心，比如「你摔倒关我什么事？下次走路看着点」（《流星花园》《一起来看流星雨》的男主人设，用嘴毒傲娇掩饰关心）。

> 🧠 **提炼**：外冷内热的三件套 = **「冷的理由 + 热的对象 + 藏的伪装」**。其中「伪装」最关键——热不能直给，要藏在「顺手」「嘴毒」背后，读者自己挖出来的糖才甜。

## 第二步：细节填充让人设落地

### 「冷」怎么写
- **少用形容词，用具体行为代替**：
  - ❌ 错误示范：他冷冷地看了她一眼。
  - ✅ 正确案例：他翻书的动作顿了顿，目光扫过她时像掠过陌生人。
- **制造距离感**：喜欢独处，与人保持距离；不喜欢社交，拒人于千里之外。

### 「热」怎么写（藏着写）
| 手法 | 范例 |
|------|------|
| **暗中观察** | 女主感冒时，他把感冒药放在她桌上；发现女主爱看某本书，悄悄把绝版版本放在她工位 |
| **身体语言** | 女主靠近时，他耳尖发红却偏头躲开；女主危险时，他下意识伸手挡在她身前，又迅速收回 |
| **嘴硬心软** | 女主加班到深夜，他皱眉：「效率这么低？明天别迟到。」但默默在楼下给她叫了车 |

> 🧠 **提炼**：「热」的全部技巧就是**「做一套、说一套」**——手上做着最暖的事，嘴上说着最冷的话。耳尖发红、迅速收回的手、默默叫的车……这些「藏不住的破绽」才是外冷内热的糖点。

## 第三步：情感递进的 5 个阶段

| 阶段 | 表现 |
|------|------|
| **① 初见** | 女主主动搭话，他只点头或用单字回应，几乎不看她 |
| **② 关注** | 女主被误解时，他突然开口澄清；女主摔倒，他伸手要扶又僵在半空 |
| **③ 试探** | 找借口送女主回家；下雨天默默把伞推向她那边，自己淋湿半边肩膀 |
| **④ 失控** | 女主遇险，他第一次情绪爆发，失控怒吼：「谁准你这么冒险的？」事后又因暴露脆弱而回避 |
| **⑤ 坦白** | 深夜长谈时，鼓起勇气坦白心声 |

## 四、注意

- **反差太突兀时，用小细节铺垫**：比如他总偷偷保存女主送的小物件，暗示早有好感。
- **不要情感转变太快，分阶段埋线索**：从不拒绝到主动，至少用 3-5 个事件推进。

## 五、经典场景（可直接套用）

| 场景 | 写法 |
|------|------|
| **办公室** | 女主熬夜做的方案被否定，男主路过扫了一眼，当晚发消息给女主修改 |
| **冲突爆发** | 女主被反派纠缠，他出现时一脸不耐烦，解决坏人后，冷着脸扯下领带帮她包扎伤口 |
| **日常细节** | 女主随口说想吃某家店的蛋糕，第二天办公桌上莫名出现一盒 |

> 🧠 **总公式**：先设定清晰的逻辑（为什么冷/对谁热/怎么藏）→ 再用具体细节填充 → 让冰山男主既有距离感又不失人情味。**让读者慢慢发现男主的温柔，比直白描写更有吸引力。**

### 🧠 整理思考

**结构判断**：塑造法型。原文结构与 [[三步塑造高岭之花]] 完全同构——「三问 / 细节 / 五阶段 / 注意 / 经典场景」。我保留原结构，把「冷的理由」「热的写法」「五阶段」「经典场景」各收成表格，并加「🧠提炼」。

**知识提炼**：
1. **三件套 = 冷的理由 + 热的对象 + 藏的伪装**，伪装最关键。
2. **「热」= 做一套说一套**，藏不住的破绽才是糖点。
3. **糖要读者自己挖**——慢慢发现比直白描写更上头。

**使用建议**：
- 甜宠/暗恋/救赎向男主首选写法；经典场景可直接套用。
- 与 [[三步塑造高岭之花]] 互为姊妹篇，按角色是「高贵冷」还是「创伤冷」选用。
- **坑**：外冷内热最怕「热得太快」或「冷得没理由」。前者靠分阶段埋线索，后者靠给冷漠一个创伤/环境/性格的根。

**关联**：定义层见 [[高冷禁欲人设]] [[阴冷孤僻人设]]（外冷内热的「冷」多来自这两类底色）。塑造工序姊妹篇 [[三步塑造高岭之花]]。与 [[傲娇毒舌人设]] 的「嘴硬心软」伪装手法同源。
