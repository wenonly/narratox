> **一句话**：情节结构四大技法合一——用「七步情节细纲」搭情节，用「伏笔」埋线索，用「铺垫」蓄高潮，用「节奏」控起伏，把平淡流水账写成让人追看的爽文。
>
> **写作价值 / 适用场景**：这是网文「好看与否」的核心引擎。情节是骨架、伏笔是暗线、铺垫是蓄力、节奏是呼吸——四者配合，故事才有起伏有期待。适合所有作者系统掌握情节结构。

---

## 一、情节设定（七步细纲）

**情节 = 小说中的人物在不同环境下应对的各种事件。** 完整情节含开端、发展、高潮、结局。情节与人物两种关系：通过人物构建情节，或为推动情节塑造配角。

### 情节细纲七要素（填表即成情节）

| 要素 | 自问 |
|------|------|
| **① 地点环境** | 这个情节发生在哪？场景有什么特点？ |
| **② 主要人物** | 涉及谁？以什么方式登场？各承担什么角色（提供线索/解谜/助力/阻力/被踩/被装逼）？ |
| **③ 故景梗概** | 要展现什么（讲故事/引人物/解谜/展性格/展实力）？ |
| **④ 高潮爽点** | 最大爽点是什么（奇遇/升级/寻宝/泡妞/发财/欺人/助人/探奇/求知）？主角得到什么好处或成长？ |
| **⑤ 渴望** | 各主要人物内心渴望什么？（按人物分列） |
| **⑥ 阻力** | 阻碍各人物渴望的因素？（按人物分列） |
| **⑦ 行动 & 冲突** | 角色为达目标如何行动？产生什么冲突？结果如何？ |

「渴望 + 阻力 = 冲突」是情节的发动机（与《网文入门与签约指南》的开篇冲突公式一致）。每段情节都该有明确的渴望、阻力、冲突、爽点。

### 情节三注意
1. **真实自然**：人物行动必须符合其性格。违反性格的情节会招反感（如让克制自守的唐僧沉迷女色 = 人设崩塌）。**再有新意的情节都不应脱离人物性格。**
2. **围绕人物性格和命运展开**：人物是情节中心，读者真正关心的是人物性格命运。长时间偏离 = 水情节，重则弃书。
3. **男女频禁忌**：「**女怕种马，男怕绿帽**」——网文感情线雷点。男频主角被绿帽、女频男主到处沾花惹草，都会激怒读者（月关早年写女主出轨被喷，直到《回到明朝当王爷》才翻身）。归根到底是代入感。

## 二、悬念 / 伏笔 / 铺垫（三者区分 + 伏笔技法）

三者都为增强读者期待感，但各不相同：

| 技法 | 作用 |
|------|------|
| **悬念** | 吸引追看（另文专讲） |
| **伏笔** | 避免剧情突兀、增强合理性；让读者发现时产生智力快感 |
| **铺垫** | 为高潮蓄力，使来龙去脉自然 |

### 伏笔（为重要剧情提前设提示，到时呼应让读者恍然大悟）

**为什么要埋**：避免「天降神兵」的突兀——前面有交代，逻辑更严谨、读者更易接受；有经验读者把发现伏笔当智力快感。

**怎么埋**（可用人/物/桥段/对话）：
- **人物/桥段**：《琅琊榜》反复出现的「野人」其实就是被谋害的赤焰军前锋聂峰（反复被提到但「死了」的人最后起死回生）。
- **物品**：主角得到的物品起初无特别，后来强大（《传奇》的麻痹戒指）。
- **对话**：换地图前几章就提到那个地方，而非突然换。

**四条注意**：
1. **反复强调**——让读者注意到不同寻常，但不马上给答案，引发期待。
2. **有伏必有应，且埋设与呼应不能脱离太久**——否则读者记忆消失，伏笔失去意义。
3. **不要让伏笔单纯当伏笔**——它还应承担其他作用（如凶手的尖锐雨伞：伏笔→凶器→破案线索）。
4. **伏笔不应过多**——过多会搞不清状况、主题模糊（除非连贯性伏笔）。

**埋伏笔的前提是对剧情走向有全盘把握**——所以要在写大纲时就设置好伏笔点和解决时机。见《小说大纲写作总集》。

## 三、铺垫（为高潮蓄力，没铺垫就没高潮）

**铺垫 = 在人物出场前/事件发生前，为显示性格或暗示命运、增强高潮必要性和真实感，而安排的非主要情节或细节。** 本质是烘托手法。

**怎么铺（从三要素入手）**：

| 类型 | 技法 | 名例 |
|------|------|------|
| **人物铺垫** | ① 提前几章甚至几十章挖坑暗示（半路杀出的人物没铺垫会突兀）；② 牛逼人物用传闻/他人对话/另一个牛人衬托出场 | 金庸《笑傲》：东方不败出场前，田伯光、令狐冲、任我行都夸他厉害；令狐冲前五章都从他人眼中勾勒 |
| **情节铺垫** | 重要情节爆发前必有铺垫。装逼打脸前铺垫：主角低调、隐藏实力、双方仇恨、反派耀武扬威、吃瓜群众围观——都是为打脸更爽 | 《甄嬛传》华妃之死：大量篇幅铺她飞扬跋扈却独宠皇上、皇上只因她哥才宠她且让她怀不上，悲剧性层层铺垫至爆发 |
| **气氛铺垫** | 用环境描写（自然/社会）渲染氛围。恐怖情节前渲染恐怖气氛，悲剧前渲染悲伤（或以乐景衬哀情） | 《水浒》武松打虎前「日色渐渐坠下去」暗示紧张；林冲风雪山神庙「那雪正下得紧」为阴谋蓄势 |

网文行话：「**越是老白的小说，铺垫越多；越是无脑爽文，铺垫越少。**」善用铺垫 = 作品成熟度的标志。

## 四、节奏（高低起伏 + 详略得当）

**节奏两重含义**：① 情节的高低起伏、轻重缓急；② 单位字数内叙述的情节数量（同样情节，篇幅多则慢，少则快）。

### 两种节奏病
| 病 | 表现 | 药 |
|----|------|----|
| **流水账** | 没起伏，有什么记什么，无聊 | 有选择地记录，剪掉平淡铺垫，选取能产生矛盾冲突的故事（**文似看山不喜平**） |
| **处处高潮** | 持续紧张，没铺垫没缓冲，读者累、易崩文 | 学会铺垫 + 注水，穿插轻松情节（如悬疑剧偶尔插主角吃饭浇花） |

### 男女频节奏差异
- **女频**：无明显等级体系，节奏像**心电图**（高低循环），遵循「三章一小高潮、五章一大高潮」。
- **男频**：有升级系统，节奏像**阶梯式**（一步步往上），但同样要有缓和情节，不能一直爆发。可学番茄（辰东）的书、《重生之都市修仙》的情节安排。

### 调整节奏 = 注水 / 放水
- **太快（要减缓）→ 注水**：一句能说清的用一段话写。
- **太慢（要加快）→ 放水**，三招：
  1. **直接删不重要场景**（「谋杀你的爱人」——为简洁忍痛割爱）。
  2. **合并场景**（如把「男主得癌离开 + 女主发觉寻找 + 崩溃打听真相」三四个场景，合并成「男主留信离开，朋友直接跟女主说明」）。
  3. **用叙述代替描写**：「林双拿到诊断单，是癌症。他觉得自己无法面对深爱他的女友。」比大段心理动作描写更简洁、节奏更快。
