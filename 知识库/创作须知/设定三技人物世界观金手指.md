> **一句话**：网文设定的三大支柱合一——人物怎么立得有戏、世界观怎么搭得自洽、金手指怎么开得不崩，一篇讲透「设定」这门手艺。
>
> **写作价值 / 适用场景**：设定是网文的底盘——人设决定代入感、世界观决定真实感、金手指决定爽感。三者为「小说设定」的三块基石（第三块情节见《情节伏笔铺垫节奏》），适合动笔前系统搭设定时参考。

## 一、人物设定（最重要的设定）

**小说设定分三类：世界观、人物、情节。其中人物最重要**——剧情转折、戏剧冲突、场景转换都跟着人物走；故事主线与主要人物的目的一致。

### 人设是什么
为更好塑造角色而对人物做的规划，含**外貌特征 + 性格 + 成就此人物的生活背景**。小说写作实际上从人物设定开始。

### 人设从何而来（原型法）
推荐《经典人物原型 45 种：创造独特角色的神话模型》（维多利亚·林恩·施密特）——把文学影视人物总结成 45 种原型（男主/女主/反派/配角），原型是「构造栩栩如生人物的蓝图」。书末《女性和男性的旅程》对性格心理刻画很有启发。（注：本知识库《人设档案》的 27 个具体人设，就是原型法的本土化实例库。）

### 怎样做好人物设定（3 个让角色有戏的技巧）

一个过于普通的角色（如「喜欢上网聊天的宅男」）吸引不了读者。让角色有戏的三步：

**① 为角色添加冲突**（万能公式：**冲突 = 渴望 + 阻碍**）
> 例：小明渴望认识更多人（尤其女性），但性格极度内向、有社交恐惧——他只能在网上外向风趣，现实中连和陌生人说话都紧张。**渴望 × 性格阻力 = 看点**，读者会想知道他能否突破。

**② 设好冲突后，多问几个「为什么」**
> 为什么他有这种渴望？为什么形成这种性格？（小时候如何？家庭如何？）从上往下倒推角色的经历，直到出生、家庭背景甚至父母的故事——角色立刻丰满，从纸面路人变活人。

**③ 多做几个「假如」**
> 假如网聊的美女突然来到他的城市约见面，他会如何？假如网上的朋友在他家附近遇麻烦，他会如何？——根据角色性格推演情节，把角色丢进「麻烦/为难/纠结」的情境（冲突），戏剧性情节就产生了。

这三步（加冲突 → 问为什么 → 做假如）是人物设定的黄金流程：**冲突让角色有戏，追问让角色有根，假让让角色驱动情节。**

## 二、世界观设定（故事的公理）

**世界观 = 故事背景 / 自然法则，类似数学的「题设」或公理——一旦设定，就是这个世界假设成立的客观规律。**（如俄罗斯方块的世界观：天空会掉四联宝石、填满一层就消失、触顶世界毁灭。）

要求：**逻辑自洽 + 足够支撑一部小说的情节发展。**

### 经典世界观流派（可借鉴）
| 流派 | 开创/代表作 | 关键设定 |
|------|-------------|----------|
| **修真类** | 《蜀山剑侠传》发扬、《缥缈之旅》奠定 | 人界（世俗界+修真界）→ 化神期飞升灵界 → 仙/神/妖/魔/鬼界 |
| **洪荒流** | 《佛本是道》（架于《山海经》《封神榜》） | 大道/天意/鸿蒙；修炼目标成圣（天道圣人）；盘古三清、西方二释、圣母娨皇六圣；大道（鸿钧）最高 |
| **无限流** | 《无限恐怖》 | 未知科技创造独特空间，把现世人召唤进去，送往电影/动漫/游戏的虚拟空间历练 |

### 新手怎么设定（4 条原则）
1. **基于现实世界微创新，而非构建全新世界**——好处是读者易代入，且世界观一旦变化对现象影响巨大，全新世界易把自己绕进去、给读者挑错。《龙蛇演义》开创国术流就是基于民国武林。
2. **做足够的储备阅读**——确认类型后，找几部同类最火的看一遍了解轮廓，再加新想法。（但别查太多资料，会忘了「创造世界」的初衷。）
3. **从细节入手**——不生硬解释设定，而从日常物品带读者进入。如《修真世界》的「音圭」（修界版收音机）：拿起来注入灵力就能收听，比硬讲设定自然。
4. **把设定融入情节**——不大段介绍，而在情节中带出。《佛本是道》把「现代修道者要买矿炼器、灵药被收刮干净」的设定融在主角赶路的情节里。

### 世界观基本要素（两大模块）
- **自然**：地理位置、地形（山脉/河流/沙漠）、植物气候。
- **人文**：国家政治结构、经济体系（钱币）、社会阶层、职业系统、种族/民族、宗教/文化。

## 三、金手指设定（开挂的艺术）

**金手指 = 为小说人物赋予别人没有、不易获得的天赋或状态**（源自游戏修改器，即开挂/作弊）。网文中几乎必不可少。

### 金手指分类
| 类型 | 说明 | 例 |
|------|------|----|
| **有形/外部** | 非天生，外界意外获得（法宝/秘笈/导师） | 《凡人修仙传》小瓶子、《仙逆》石珠、《斗破苍穹》药老 |
| **无形/内部** | 天生拥有的能力（经验记忆/资质异能/血统体质） | 穿越重生的记忆；《遮天》荒古圣体；《帝道至尊》大势力的继承人兼天赋惊人 |
| **系统流/特殊属性** | 系统或剧情安排（关键时刻突破、大难不死） | 《超级怪兽工厂》纯靠系统推动 |

### 金手指的三大作用
1. **满足 YY 让读者爽**：现实成马云难于上青天，但重生/穿越一次（金手指）就能做到——读者代入「靠机遇运气得来的成功」比「纯奋斗」更爽。
2. **推动情节发展**：金手指的根本用途。如斗破——主角需药材炼药提升修为，找药材的过程就发展出丰满情节。**金手指为推动情节而存在，不是单纯为爽**（为爽乱开 = 假大空，读者不爽）。
3. **提升读者期待感**：主角天资平庸凭什么能反转？面临危机凭什么能脱险？——靠金手指坚定读者信心。

### 合理金手指的 5 个特点
| 特点 | 说明 |
|------|------|
| **① 唯一性** | 只主角一人拥有，显示独一无二、增加主角光环 |
| **② 拓展性** | 必须可升级，才能不断推动情节、给期待感；万能金手指也能靠升级体系保持平衡 |
| **③ 限制性** | 不能一开始就强大（否则后期崩文）。如小瓶子有时间限制。跟着节奏：主角需大成长时开大一点，从简单到复杂 |
| **④ 保密性** | 最好除读者和作者，书中没第二人知道（增强主角安全 + 爽点） |
| **⑤ 简单性** | 别设计太复杂，让读者一目了然，易形成完整概念 |

### 注意事项
- **不宜开太多太大**：金手指一多就丧失特色；**爽点不在多，在于金手指本身能升级**。循序渐进才能稳定留读者。
- **不能依赖金手指**：网文核心是主角个人能力提升，金手指只为主角服务；最好的金手指是能跟随主角成长而成长。

### 金手指设定自检清单
- 主角的金手指是什么？一个还是多个？暂时性还是贯穿全书？
- 有成长性吗？（一开始是完善系统随主角成长露面目？还是残缺的随主角升级修补而进化？）
- 限制是什么？重复使用有什么副作用？
