## 一、叙述技法

### 五种表达方式
写作有 5 种表达方式：**记叙、描写、抒情、说明、议论**。小说主要用**记叙 + 描写**（其余几乎用不到）。二者的区别见 [[描写技法大全]]（叙述=笼统交代，描写=具体描绘）。

### 叙述的两大作用
1. **交代背景**：交代社会环境、时间、人物、事件起因结果等（记叙六要素：时间/地点/人物/起因/经过/结果）。写大纲时一般直接用叙述，写正文才用描写。
   > 《择天记》开头用叙述交代了西宁镇、破庙、师徒三人、大周国教背景。
2. **快速推动情节**：叙述一句话能带过的事，描写要花大量篇幅。
   > 《斗罗大陆》「霍雨浩五年刻苦努力修炼到十级」——叙述一句带过，若用描写要大段。

### 记叙的顺序
| 顺序 | 说明 |
|------|------|
| **顺叙** | 按时间先后顺序（大部分小说都是顺叙） |
| **倒叙** | — |
| **插叙** ⭐ | 叙述中心事件时，暂时中断线索，插入一段相关的回忆或故事。**起补充衬托、解释说明作用**，插叙完接着原情节，使情节完整紧凑。《何以笙箫默》插入女主认识周教授的回忆。**小说里最常用** |
| **补叙** | — |

### 对话「第二种形式」的高阶用法（突出重点）
「内容 + 提示语 + 内容」这种形式有独有优势——**突出重点**（网文读者一目十行，突出重点很必要）：

- **突出前半**：「**阴谋！**」张三眉毛皱成一团，「李四是个市侩小人……」——后面那堆塑造人物，推进情节的只有前面「阴谋」二字。**越短小的部分越突出。**
- **突出后半**：「……」张三面带微笑，一双眼睛却闪着寒光，「**有埋伏！**」——重点放最后。

**中间的描述部分也有用**：可体现人物性格（张夫人抹泪护住小姑娘），或埋伏笔（张三「正了正玉冠」可作行刺暗号）。

> 🧠 小技巧要为三要素（人物/情节/环境）服务，不能喧宾夺主。

## 二、网文行业术语表（新人必懂）

### 成绩与作品状态

| 术语 | 释义 |
|------|------|
| 扑街 | 作品发表后成绩不好 |
| 太监 | 小说连载到一半突然中断 |
| 烂尾 | 情节未圆满就中途夭折或草草收尾 |
| 崩 | 世界观/剧情出现多处 bug、体系不能自洽，情节混乱逻辑不清（常是太监/烂尾的开始） |
| 坑 | 前文某些情节（人物/事件）后文没交代 |
| 单机 | 作品没读者，只有作者自己看 |
| 切 | 作品未完结但成绩差，作者直接放弃转开新书 |
| 精品 | 订阅达一定数量（起点均订 3000），有机会被改编游戏/影视 |
| 小神/中神/大神 | 精品文作者 / 多部稳定精品且有知名度 / 成绩逆天或某分类标志性人物 |
| 中原五白 | 东茄（我吃西红柿）、西豆（天蚕土豆）、南机（梦入神机）、北少（唐家三少）、中辰东 |
| 证道 | 本意得道，引申为成功 |
| 百万必神 | 写到百万字必成神（行业吉语） |

### 签约与运营

| 术语 | 释义 |
|------|------|
| 签约 | 作品达网站要求后签版权协议 |
| 上架（入V） | 入 VIP，读者需付费才能看；是作者赚钱方式之一 |
| 内签 / 内投 | 直接联系编辑取得签约 / 主动联系编辑投稿 |
| 首订 / 高订 / 均订 | 上架第一章 24h 订阅 / 单章最高订阅 / 平均每章订阅（总订阅÷VIP 章节数） |
| 买断 / 分成 / 保底分成 | 网站一次付费任意使用 / 按读者消费比例分 / 保底买断价 + 分成 |
| 千字稿酬 | 按一千字计算稿酬 |
| 月票 | 起点首创，会员订阅打赏获投票权，象征人气与认可（每张值 100 粉丝积分） |
| 全勤 | 上架后按月不断更，网站给的福利（全勤奖） |
| 低保 | 成绩不好时（如起点月收入低于 1500）可申请的最低保障，要求每日更新一定字数 |
| 盟主 | 粉丝榜最高等级（100000 粉丝积分）；其下有宗师/掌门/长老/护法/堂主/舵主/执事/弟子/学徒 |
| 马甲 | 作者换笔名（一般指大神开小号） |
| 推书 | 在作品里给其他书打广告 |
| 站短 | 站内短消息（审核通过/签约会有消息推送） |
| 裸奔 | 没有推荐 |
| 断更 / 催更 | 上架后某段时间中断更新 / 读者催促更新（网站有催更道具） |
| 限免 | 限时免费（网站促销） |

### 推荐位（旧术语，了解即可）

三江推 / 文字推（裤衩推）/ 分强（分类强推）/ 强推（首页强推）/ 大封吹（大封推）/ 端推（手机端推荐）/ PK（一个推荐位同时上 N 本，按数据淘汰）。

### 类型与概念

| 术语 | 释义 |
|------|------|
| 金手指 | 给主角赋予常人没有的能力（超智/颜值/武功/财富/特殊能力/无敌运气） |
| 换地图 | 主角行动地点从一个换到另一个（小至城市，大至世界） |
| YY | 意淫缩写；YY 小说=实现现实中无法实现之事，情节严重不符合现实逻辑 |
| 小白文 | 情节简单、无深度、内容浅白、读来放松，读者低龄化 |

## 三、创作工具清单（按写作流程）

> 🧠 工具更迭快，以下为类别框架 + 代表，具体工具请自行验证当前可用性。

### ① 查资料
- **天人之境**（奇幻文学资料库）、**殆知阁**（历史资料库）、**萌娘百科**（二次元）、**纯美苹果园**（DnD 中文）。
- **Tvtropes**（国外影视/小说桥段数据库）、**玄派百科**（致力于中文版 Tvtropes）。

### ② 构思（思维导图）
- PC：Scapple、mindnode、**Xmind**（免费、式样多）、mindmanager、百度脑图（在线免费）。
- 手机：ithoughts、mindmeister、思维导图。

### ③ 写作（码字）
- PC：玄派写作神器、写作猫（双端同步、可同屏显示细纲角色）、Word/WPS、码字精灵（拼字/赏罚）、大神码字。
- 手机：口袋写作、小黑屋、汤圆写作、**作家助手**（阅文集团官方，可查稿费订阅）。

### ④ 云储存
印象笔记、有道云笔记等。

### ⑤ 交流学习
- **龙空论坛**（国内最大网文讨论论坛）。
- 公众号：玄派、杨晨说网文、作家助手。
- 知乎号：跳舞、流浪的蛤蟆、写手 K 君、羊驼、唐缺。

### 🧠 整理思考

**结构判断**：基础常识聚类。叙述、术语、工具是「入门基础」主题，合并成一份行话+技法+工具的速查。术语表按 2026 视角分类整理，旧推荐位术语标注「了解即可」。

**知识提炼**：
1. **叙述 = 交代背景 + 快速推动情节**；插叙是小说最常用的补充说明手法。
2. **对话第二种形式可「突出重点」**（越短越突出），中间描述可埋伏笔。
3. **术语是行话通行证**——不懂均订/首订/证道/扑街，就读不懂编辑反馈和同行交流。
4. **工具按流程选**：查资料→构思→码字→交流，每个环节都有专门工具。

**使用建议**：入行先通读术语表（尤其成绩/签约类，直接关系收入理解）。卡情节时用插叙补充说明；想让读者抓住重点时用对话「第二种形式」+ 短句前置/后置。

**关联**：叙述与描写的区别见 [[描写技法大全]]。术语里的「金手指/换地图」详解见 [[设定三技人物世界观金手指]] [[爽文套路库]]。签约术语的实操见 [[网文入门与签约指南]]。
