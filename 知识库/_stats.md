## 文档格式

- **frontmatter 极简**：仅 `tags: [...]`（不保留 source / ocr 质量 / 整理状态 / 字数等溯源元数据）。
- **正文**：开头「一句话 + 写作价值/适用场景」TL;DR → 分级标题重建结构 → 能表格化尽量表格化 → 金句原文照录 → 末尾固定「🧠 整理思考」（结构判断 / 知识提炼 / 使用建议 / 关联，用 `[[文件名]]` 互链成网）。
- 整理思考只保留有价值的判断与方法论，不含源文件名/OCR 订正清单/抽取方式等过程信息。

## 服务方式

`KnowledgeService`（`server/src/knowledge/knowledge.service.ts`）启动时扫描 `知识库/<6 分类>/*.md`：解析 frontmatter 的 `tags`、提取开头「一句话」作 description、按分类+文件名排序生成稳定 id（`zl{NN}`），内存缓存。提供 `list / getEntry / search`，经 `KnowledgeController`（`/knowledge`）给 agent-ui 浏览页与 curator 的 `search_knowledge` 工具使用。
