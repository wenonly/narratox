> **一句话**：以权力博弈为主线，靠智谋布局、利益制衡、人性博弈推动剧情——胜负的关键不是智商，而是心理博弈（主角能预判反派的预判）。
>
> **写作价值 / 适用场景**：权谋短篇标准公式。适合写朝堂/宫斗/智斗向短篇，核心是「隐忍蓄势 → 借力打力 → 一击翻盘」的谋略节奏。

## 一、人设要点

| 角色 | 设定 |
|------|------|
| **主角** | 善用阳谋，有道德底线，心中有国家和百姓 |
| **反派** | 动机复杂，聪明但三观与常人不符，为达目的不择手段 |

权谋文的魂在「**主角有底线、反派有苦衷**」——主角靠阳谋和道德立身，反派要有复杂动机（不是纯坏），结局体现因果循环。这样写出来的权谋才有厚度，不是单纯的智商碾压。

## 二、要点

1. **计谋需符合古代背景**：传信用密信，收集信息靠酒肆青楼。
2. **胜负的关键不是智商，而是心理博弈**：比如反派轻敌，主角能预判他的预判。

## 三、公式骨架（起承转合 + 字数）

| 阶段 | 字数 | 节拍 |
|------|------|------|
| **起** | 0–1000 字 | ① 开篇铺垫情绪，引入主角困境（被贬、流放、家族覆灭、遭灭族） |
| | | ② 展示反派的压迫感，暗示反派的阴谋（结党营私、企图篡位、陷害忠良） |
| | | ③ 主角表面隐忍，暗地收集情报，培养势力，结交人脉，埋藏暗桩 |
| **承** | 1000–4000 字 | ④ 矛盾升级，反派进一步打压主角（抄家、污名化、威胁至亲） |
| | | ⑤ 主角发现反派的破绽（贪污受贿、私通外敌） |
| | | ⑥ 主角设局，假意向反派投诚 |
| | | ⑦ 通过伪造书信、传播谣言、破坏计划，分离反派与盟友 |
| **转** | 4000–8000 字 | ⑧ 借力打力，借助第三方势力打压反派，坐山观虎斗 |
| | | ⑨ 反派察觉反扑，致使主角陷入绝境（遭背叛、证据被销毁、人身安全受威胁） |
| | | ⑩ 主角使出杀手锏（暗中保护的人证、未公开的证据） |
| **合** | 8000–10000 字 | ⑪ 对峙公堂或发动宫变，层层揭露反派的罪行 |
| | | ⑫ 反派倒台，众叛亲离；主角重掌权力但施行仁政 |

## 四、结构内核

```
起(困境+隐忍埋暗桩) → 承(发现破绽+设局投诚+离间) → 转(借力打力→反扑绝境→杀手锏) → 合(公开处刑+仁政收尾)
```

权谋是**「太极式」结构**——主角前期全在「退」（隐忍/投诚/让步），实则在「进」（埋暗桩/离间/借力）。转段必有「绝境反弹」（反派反扑致主角陷绝境，再用杀手锏翻盘）。合段的「仁政」是点睛——主角赢了却不滥权，呼应「有道德底线」的人设。
