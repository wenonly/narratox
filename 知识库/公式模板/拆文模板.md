> **一句话**：一份拿来就能用的「拆文模板」——按 5 个维度拆解任何一篇别人的爆款短篇，把它的优点变成自己下一篇能用的技巧。
>
> **写作价值 / 适用场景**：这是**学习型工具**，不是创作公式。适合用来拆解爆款（尤其是爽文）反向学习：读完一篇好文，照这张表填一遍，它的结构、爽点、人设、金句就都被你「拆」成了可复用的素材。

## 拆文五步表

### 1. 基础信息
| 项 | 填写 |
|----|------|
| 文名 / 作者 / 类型 | |
| 字数 / 点赞量 | |

### 2. 剧情分析

**结构与导语**
- **导语**：（摘录 + 分析它为什么抓人）
- **结构**：起 / 承 / 转 / 合（各写了什么）

**结构分析**
- 开头 300 字如何切入？
- 关键转折点（付费点）在哪？
- 结局设计是否反套路？

**爽点设计**（爽文专项）
- 打脸节点几次？间隔多远？
- 金手指设定（如何合理化开挂逻辑）？
- 伏笔与反转？

### 3. 核心拆解项（重点！）

**人物塑造**

| 角色 | 拆解维度 |
|------|----------|
| **主角** | 人设 / 反差感 / 目标 / 动机 |
| **反派** | 人设 / 动机 |
| 关系网 | 与各人物关系 |
| 功能 | 各个角色在文中的作用 |

### 4. 拆后总结
- 这篇文**最值得模仿的 1 个优点**：______
- 你**可用到下一篇的 1 个技巧**：______
