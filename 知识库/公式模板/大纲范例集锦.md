## 一、九大构成体系（玄幻大纲的标准框架，以《斗破苍穹》为范例）

这是从《斗破苍穹》还原出的一套**玄幻大纲九件套**，每一项都用斗破的设定示范，可照搬到自己的玄幻文：

| 构成 | 范例（《斗破苍穹》） | 要点 |
|------|---------------------|------|
| **① 主角** | 萧炎，穿越者。8 岁得到禁锢药老灵魂的戒指，被吸力量成「废物」，解放药老后获大量高级丹药、功法，实力突飞猛进 | 注重形象、性格、口头禅、实力；**故事以人为本，写好人物才有代入感** |
| **② 配角** | 萧薰儿（活泼少女，青梅竹马，口头禅「炎哥哥最好了」）；萧重（父亲，沉稳武者，口头禅「炎儿最优秀了」） | 每个配角都要有辨识度（性格+口头禅） |
| **③ 技能** | 八极拳（初级功法，分十重，每重一层暗劲即「一响」，主角练出十八响）；焚诀（药老教的顶级火系功法，禁咒级）；炼药（药老教的，吃丹药像吃糖） | **重点设主角技能，用来控制节奏** |
| **④ 伙伴** | 小医仙、冰皇（实力与主角相近） | 设定主角的助力，关键时刻（如杀上云岚宗）起作用 |
| **⑤ 装备** | 玄重尺（药老赠，初期极品新手装）；月甲（后期终极装，抵挡禁咒） | 可参考网游装备（屠龙刀/开天斩）设定 |
| **⑥ 冒险（主线+暗线）** | 主线一：外出游历，赴三年之约，上云岚宗击败未婚妻洗刷悔婚屈辱；主线二：消灭魂殿救药老。暗线一：家族与上古世家的联系；暗线二：成为斗帝 | 主线明、暗线埋，**暗线是后期的引爆点** |
| **⑦ 身世** | 萧家是上古家族之一，主角及家人并不知情，后期上古家族出世才揭露 | 身世是长线大坑，前期埋、后期爆 |
| **⑧ 势力** | 萧家、药老 | 搞清主角的盟友与敌对势力 |
| **⑨ 后宫** | 萧薰儿、雅妃、云韵 | 构思不同性格、不同外貌的女子，「总有一个是读者喜欢的」 |

**主线构架（范例）：**
> 测试失败被认为是废物 → 族弟羞辱（拉矛盾）→ 萧薰儿解围（拉后宫线）→ 未婚妻上门悔婚（拉开前期大矛盾及大坑，铺垫大高潮）→ 得到药老戒指 → 三年之约（前五十万字紧紧围绕这条线：打怪升级、寻宝、冒险、炼药夺魁）。

**最关键的在于主角的刻画**。主线构架不需要把每一章细化，只要设定好**关键情节路线、关键人物、关键道具/功法的出现节点**，随时提醒自己该写什么。**并要点出关键的「坑」，随时提醒自己记得填，以免写到后面忘了前面的伏笔。**

## 二、四环节构思法（新书怎么从零构思）

不知道写什么时，先用经典套路搭骨架，再在**第二环节**做变化：

| 环节 | 武侠经典套路 | 是不是构思重点 |
|------|--------------|----------------|
| ① 开局 | 灭门惨案，主角出逃 | ❌ 不是重点（全家被杀还是只死老爸，无关大局；穿越是被马桶冲走还是被车撞，都是表象） |
| ② 获能 | 坠崖不死，学成神功 | ✅ **构思重心**（主角学到什么本领，决定全书基调和后续走向） |
| ③ 闯荡 | 艺成出山，巧遇侠女，锄奸惩恶 | ❌ 不是构思重点（它受制于第②环节，篇幅最大但由②决定） |
| ④ 结局 | 大仇得报，武林称雄 | ❌ 不是重点（书都要结束了，结局再变也只是结局） |

**构思重心永远在「主角获得什么能力」**——这一项决定全书基调（学降龙十八掌→当大侠；学凌波微步→见人就跑）。所以新书构思时，把精力花在设计主角的核心能力/金手指上，其余环节用老套路填即可。

## 三、各频道范例的结构差异

| 范例 | 频道/类型 | 大纲特色 |
|------|-----------|----------|
| **斗破苍穹 / 男频玄幻** | 男频·玄幻 | 九大构成体系（见 §一）；力量体系+等级体系（斗之气→斗帝）是核心 |
| **男频都市异能** | 男频·都市异能 | 主线+暗线（暗线多达 5 条，每条都是后期反转）；宏大世界观（如「九大国度」设定）；预计 300 万字 |
| **女频大纲（《樱花殇》）** | 女频·古风 | **人物网是核心**——每个角色都有身份、性格、与前因后果；结局先定（谁死、谁牺牲、如何反转）；「我即是顾北辰」的身份反转贯穿全局 |
| **西游记大纲** | 经典名著 | 梗概式——按时间线把大事件串成流水（石猴出世→大闹天宫→被压五行山→取经缘由→师徒集结→九九八十一难）。适合做「长篇史诗」的大纲范本 |

## 四、从范例提炼的大纲共性要点

1. **故事以人为本**：写好人物（形象/性格/口头禅/实力），平凡剧情也能精彩。
2. **主线构架只设关键节点 + 关键的坑**，不必逐章细化，但要随时提醒自己填坑。
3. **暗线是长篇的发动机**：前期埋（身世/家族秘密/隐藏身份），后期引爆。
4. **力量体系要自洽**：写清原理来源（方便描写修炼和「偶然突破」的合理性）+ 每级差异（获得哪些技能）。
5. **结局先定**（尤其女频/悲剧），再倒推前面铺垫。
