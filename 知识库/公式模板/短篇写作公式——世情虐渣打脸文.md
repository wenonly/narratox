> **一句话**：开篇用倒序+五感+记忆闪回把女主的惨堆到顶，每节结尾埋小钩子，再用「降维打击」式的复仇一步步反杀渣男小三——爽文的究极形态。
>
> **写作价值 / 适用场景**：世情虐渣打脸文（现代家庭伦理爽文）标准公式。适合写出轨/复仇/女强逆袭向短篇，是当下短篇市场最吃香的题材之一。

## 一、要点（爽文的 4 个加速器）

1. **开篇情绪叠加**：结合倒序、五感、记忆闪回。例如——看到小三在朋友圈秀钻戒；丈夫说「娶个花瓶最大的好处就是钱都是自己的」；亲戚朋友的异样目光和偷笑；自己悲惨的处境；父亲的遗嘱和遗言。
2. **每节结尾设置小钩子**：例如「没人知道，我的手机正在同步直播丈夫和小三的婚宴，包括他非法转移资产的证据。」
3. **主角反击加入「降维打击」，让复仇更爽**：女主用法律知识把小三告上法庭；用财务知识看破丈夫的阴阳合同。
4. **大结局要物质 + 精神双丰收**：夺回家产，成为女强人后被众人爱戴。

> 🧠 这 4 条是这个题材的爆款密码：① 开篇抓人（情绪叠加）；② 节末留钩（追更率）；③ 降维打击（爽点升级）；④ 双重收获（爽感闭环）。

## 二、公式骨架（起承转合 + 字数）

| 阶段 | 字数 | 节拍 |
|------|------|------|
| **起** | 0–1000 字 | ① 开篇描写主角丈夫出轨并当众羞辱她，铺垫情绪 |
| | | ② 情绪叠加：描写反派和小三的丑恶嘴脸 |
| | | ③ 情绪叠加：描写女主的惨状、精神状况 |
| | | ④ 情绪叠加：记忆闪回，用主角多年的付出和对感情的一心一意，对比反派的丑恶 |
| **承** | 1000–4000 字 | ⑤ 复仇行动一：伪装身份接近小三，解决小三 |
| | | ⑥ 复仇行动二：借刀杀人，解决反派的背景或家族 |
| **转** | 4000–8000 字 | ⑦ 复仇行动三：请君入瓮，解决反派 |
| | | ⑧ 复仇行动四：反派为自保出卖家族产业，陷入主角的局中局 |
| | | ⑨ 小三反水，曝光反派做的恶心事 |
| | | ⑩ 反派的家族陷入危机，决定让反派自生自灭 |
| **合** | 8000–10000 字 | ⑪ 反派净身出户，遭所有人抛弃；主角拿回属于自己的房产股份 |
| | | ⑫ 主角事业成功，成为首屈一指的事业女强人，被邀请回母校演讲 |

## 三、结构内核

```
起(情绪叠加 4 连，攒恨+立惨) → 承(解决小三+反派家族，剪羽翼) → 转(请君入瓮+局中局+小三反水+家族抛弃，连环崩塌) → 合(净身出户+女强逆袭，双丰收)
```

> 🧠 这是「**剪羽翼 → 打主体 → 连环崩塌**」的三段式清算：承段先解决小三和反派家族（断其外援），转段用局中局逼反派自保卖产、小三反水、家族抛弃（连锁反应），合段反派净身出户、女主双丰收。爽感来自**层层升级的「社会性死亡」**。

### 🧠 整理思考

**结构判断**：公式型。12 个节拍收进表格，要点 4 条单列（这是本文最值钱的部分）。

**知识提炼**：
1. **爆款四加速器**：情绪叠加开篇 + 节末钩子 + 降维打击 + 双重收获。
2. **「降维打击」是现代爽文的核心爽点**——用专业知识（法律/财务）碾压，比情绪化报复更高级、更解气。
3. **清算要层层升级**：剪羽翼 → 打主体 → 连环崩塌，社会性死亡最大化。

**使用建议**：开篇直接用「朋友圈秀钻戒+丈夫金句+亲戚目光」三连把惨立住；复仇用「降维打击」（给女主一个专业身份：律师/会计/医生）；结局务必双丰收（物质+精神）。换皮：职场/校园/婆媳都能套同结构。

**关联**：与 [[短篇写作公式——复仇文]] 是姊妹篇（都先虐后爽），区别：复仇文偏「暴力清算 + 道德替天行道」，世情虐渣偏「智斗降维 + 社会性死亡」，现代感更强。可叠加 [[短篇——追妻火葬场模板]] 的「女主决绝 + 事业开挂」设定。
