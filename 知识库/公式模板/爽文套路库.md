## 一、现言宠文类

| 维度 | 设定 |
|------|------|
| **标签** | 总裁霸爱、男神我嫁、豪门联姻、契约恋人、大宠小虐、宝宝、下药、替身、娱乐圈 |
| **类别/背景** | 豪门恩怨 / 总裁豪门 |
| **男主** | 总裁（优质好男人，男神形象） |
| **女主** | 草根、没落千金（身份随意，**讨喜是关键，性格独立是亮点**） |
| **主线** | 接地气「先婚后宠」：协议结婚，日久生情，男女主三观端正、都较独立。可设配角甲乙丙丁、前女友前男友增加波折，但**男女主不可出轨**，只是调味 |

### 雷区（4 条铁律）
1. 男主不宜风流——走「外表邪魅狂狷、内心稳定可靠」，必须有 hold 住一切的气场，绝对维护女主。
2. 男主不宜在和女主产生感情后与女配发生关系，不宜与女主之外的女性生孩子，**不可为虐而虐**。
3. 男女主关系正常，**不宜乱伦**（叔嫂、姐夫和小姨子尤其避免；一方已婚前提下绝对避免）。
4. 不宜纠结过多日常琐事（平淡流水账缩减），只写能带动情节或感情发展的桥段。

### 建议设定
- 多增加暧昧桥段，推进感情深入。
- 女主可设身份之谜，增加主线，但人物关系要清晰。

### 参考范例（3 种开局）
1. **一夜情流**：一夜情 → 契约婚 → 日久生情 → 前女友捣乱 → 女主受挫/反击 → 男主维护 → 女主身世 → 男主帮解决（或误会分开）→ 历经挫折幸福一起。
2. **家族联姻流**：家族联姻 → 男主仇恨（可设对女主母家的仇恨）→ 实际动心 → 前女友/前男友制造挫折 → 男主被折服正视真心 → 勇敢追 → 幸福一起。
3. **错认流**：错误相亲（认错人）→ 火速领证 → 搞定双方家庭 → 日常暧昧生情 → 前任制造波折 → 男主从一而终 → 女主身世波折 → 解决幸福一起。

> 参考作品：《先婚厚爱》《降服高官老公》《总裁的私有宝贝》

## 二、重生复仇爽文类

| 维度 | 设定 |
|------|------|
| **标签** | 重生复仇、阴谋心计、扮猪吃虎、爽文无敌、豪门恩怨、世家情仇、宅府深院、庶女、嫡女、丑颜 |
| **类别/背景** | 古代言情、穿越架空、豪门恩怨 / 古代大家族、现代豪门 |
| **男主** | 强大神秘，只对女主好 |
| **女主** | 重生后聪明擅长心计，手段果决 |

### 主线设定（5 条）
1. 女主前世被宠坏或不受宠，被陷害致死、死得很惨，重生复仇、扭转命运，整治渣男渣女，重新收获幸福。
2. **重生到一切还来得及扭转的时候即可，不要重生到小时候**；重生后必须变强或找到强大后盾，保证立于不败之地。
3. 爽点要直接明白——对渣男渣女的复仇反击要有力，不用一击致死，但要造成很大损失。
4. 明确敌我阵营，**女主靠山不可动摇或被反派挖走**（会降低爽点）。
5. 设多个反派，**等级逐渐增强**，制造爽感的升级和层次。

### 雷区（5 条）
1. 不宜重生到小时候或重生去奋斗（易变种田文）。
2. 不宜和前世害过女主的渣男纠缠（前世爱情不作数，今生重来）。
3. **重生后的男主不能误会女主、不能开虐**（爽文不能有虐点）。
4. 反派层级从家宅开始，但**不可局限于一家一室**，女主最终要走出去（否则变宅斗）。
5. 不能连续好几章不给反派制造伤害（节奏太慢；反击要快而狠）。

### 建议设定
- **家内反派标配**：渣姐/渣妹 + 继母/嫡母 + 趋附恶势力的下人 + 偏帮渣女和继母的父亲。
- 女主重生后立于不败之地，看渣渣们扑腾，一个一个收拾（先抑后扬、扮猪吃虎，不可过于张扬）。
- **拉高反派仇恨值**，让渣渣再渣一点，女主抽得更爽。

### 参考范例
前世被蒙蔽/不受宠凄惨死去，死前得知真相 → 重生到能扭转的时期，变强 → 扮猪吃虎，小范围收拾身边下人，收获亲信/靠山 → 等对方下手并反扑 → 与家内反派斗争，洗脱前世污名，让恶势力公诸于众 → 与欣赏女主的人结盟，获得际遇、好名声、高身份压制恶势力 → 收获全能好男人（身份本领高于一切男配），并肩作战 → 铲除家内+男主的对立面，陪男主走向至尊（复仇线可穿插奋斗线、感情线；适当换地图引入高级反派增加层次，女主可被适当压制但终立于不败）。

> 参考作品：《庶女有毒》《侯门毒妃》《重生豪门千金》

## 三、穿越女强类

| 维度 | 设定 |
|------|------|
| **标签** | 无敌升级、废柴翻身、至尊宠溺、特工穿越、神医逆世、扮猪吃虎、魔宠、元素召唤师 |
| **类别/背景** | 穿越架空、玄幻女强 / 架空朝代、异界大陆 |
| **男主** | 神秘强大，性格优质，只对女主好 |
| **女主** | 由弱变强，惊艳翻身，最好带技能或超常天赋 |

### 主线设定（5 条）
1. 穿越前弱女死去，现代强势女主穿越过来，性格由弱变强，改变凄惨命运。
2. 女主带医术/武力/其他天赋，在古代有超常发挥。
3. 玄幻体系要**简洁易懂的等级**，女主升级速度远超常人，或给超常体质。
4. 玄幻类可加魔宠（卖萌兼协助打怪）。
5. 爽点：废材翻身、越级抽傻逼、升级快、女主无敌、众男围绕、男主痴情。

### 雷区（5 条）
1. **不宜设虐点**（只有女主虐别人的份）。
2. 男主不宜风流下流、有前任（即使有也必须立即见识到女主的好后被迷上）。
3. 女主变强但不能光谈恋爱忽略抽傻逼的爽感。
4. **不宜局限于一个地图**，建议多开地图扩充升级可能。
5. 女主不宜目中无人（强势要有道理，抽傻逼要够狠够到位）。

### 建议设定
- 强 = 个性脾气独立 + 爱憎分明 + 本领强悍。
- 爽要直接到尾，抽得快狠准，不能不痛不痒。

### 参考范例（玄幻穿越女强）
大家族废材不受宠被折磨忽视致死 → 带技能重生天赋觉醒 → 翻身逐级抽傻逼（扮猪吃虎，外人忽视、女主一鸣惊人）→ 境界一日千里，际遇男主，收获高阶魔兽（制造不平衡感）→ 增加职业设置突破规则（普通人修一种职业，女主修多种）→ 打副本收获技能装备 → 开学院线（当副本，抽完家里傻逼后进入）→ 完结一个地图 → **换地图从高到低再来一轮**（原地图无敌后换地图重新奋斗飙升）→ 和男主并肩于不败之地。

> 参考作品：《皇后逆天斗苍穹》《天才儿子腹黑娘亲》《第一庶女》
