> **一句话**：把小说里的打斗场景拆成 7 种经典情境公式——每种都给「前期铺垫 → 造势 → 开打 → 装逼反转 → 观众反应 → 收尾」的完整节拍，照着填就能写出有层次、有爽感的打斗。
>
> **写作价值 / 适用场景**：这是**打斗场景的写法工具箱**——很多人会写打斗却写得干（只写招式不写气氛）。本篇的核心价值不在 7 种情境本身，而在贯穿 7 种的**「打斗节拍公式」**：打斗好看的秘诀是「写人（双方心理+观众反应）」远多于「写招」。

---

## 一、打斗的通用节拍公式（7 个情境共享，最该学的部分）

不管哪种打斗，好看的打斗都按这套节拍走：

| 节拍 | 写什么 | 作用 |
|------|--------|------|
| **① 前期铺垫** | 天气（写景造气氛：阴沉/压抑）、观众（亲友打气/全城空巷）、双方心理（怒火/轻视/紧张） | 造势，把期待拉满 |
| **② 出场造势** | 主角用最牛身法出场，向长辈打招呼，长辈眼前一亮赞叹；对手冷哼登台，引发双方粉丝对喊，全场沸腾 | 用「旁人反应」烘托双方分量 |
| **③ 高台议论** | 高台上的大人物们议论纷纷（一长者夸对手、一长者留悬念、宗主看好主角） | 侧面烘托 + 埋悬念 |
| **④ 开打** | 主角先一招逼退对手，**装逼淡笑**，连环招出，对手惊讶想反攻却腾不出手 | 主角的「游刃有余」是爽点 |
| **⑤ 对手反应** | 惊讶、狼狈、被打飞武器/被指着要害 | 用对手的窘迫反衬主角 |
| **⑥ 观众反应** | 全场呆了、倒吸凉气、疯狂议论、技术流观众分析招式精妙 | **观众是打斗的放大器**——他们的震撼让主角的强被无限放大 |
| **⑦ 收尾** | 点出主角境界（让宗主判断/期待）、讥讽威胁对手、或虽败犹荣 | 升华，留余韵 |

**打斗好看的核心秘诀：写「人」不写「招」**。一套打斗里，「双方心理活动 + 观众/大人物反应」的篇幅应远多于招式描写。观众是放大器——他们的呆、凉气、议论、分析，比直接写「主角很强」有效十倍。**没有观众反应的打斗 = 两个人干架，没人在乎。**

## 二、七种打斗情境（差异在「关系」和「反转方式」）

| 公式 | 情境 | 核心看点 |
|------|------|----------|
| **① 公平对决** | 对手约战，主角应战，双方势均力敌 | 正面硬刚，主角游刃有余、最后关头收招（显境界） |
| **② 主角挑战** | 主角长期被打压，练功有成来找对手教训 | 对手轻视口出狂言 → 被打脸；主角出言讥讽、点丑事威胁 |
| **③ 巅峰约战** | 对手极强（巅峰强者），全城空巷，数百万观众窒息 | 绝招对绝招、天塌地崩；**主角虽败犹荣**（和巅峰强者打成平手已是震撼） |
| **④ 翻身第一战** ⭐ | 主角极菜（陪练角色），没人看好，对手找他找回自信 | **经典翻身**：对手冲上来想秒人 → 被主角一招奇招反秒 → 全场傻眼 → 一战成名 |
| **⑤ 密训复出** | 主角小有名气后消失密训，回来挑对手 | 以静制动，用同样招式比对手更快更高更强，秒之 |
| **⑥ 名声大噪战** | 主角名声大噪，对手奸猾用赖皮招（隐身/干扰） | 主角先处下风（武器被爆）→ 借机捕捉位置反杀 → 揭示主角一开始就发现了对手（扮猪吃虎） |
| **⑦ 顶级装备战** | 对手穿顶级装备、很强很嚣张，环境对主角不利 | 主角在行动中适应环境，夺对手武器、点爆远程攻击、一记精准远程干掉 |

**公式④（翻身第一战）是打斗里最经典的爽点模板**——「人人看不起的弱者，一招反秒强者」。这个「弱胜强 + 全场震惊」的反转，是升级流网文反复用的核心爽点，务必掌握。

## 三、打斗的 5 个写作技巧（贯穿七公式）

1. **装逼淡笑**：主角开打时先一招逼退，然后「装逼的淡笑」——游刃有余的姿态本身就是爽点。
2. **心理活动穿插**：开打中写双方心理（主角的盘算、对手的惊惧），让招式有「智斗」感。
3. **观众分层反应**：普通观众（呆了/欢呼）+ 技术流观众（分析招式精妙）+ 大人物（判断境界），三层反应立体烘托。
4. **「扮猪吃虎」反转**：主角看似处下风（如公式⑥），实则一开始就掌控全局，反转一刻爽感炸裂。
5. **环境与装备的利用**：用环境劣势（公式⑦）或装备劣势（公式⑥）制造「先抑」，让反转更燃。

## 五、补充：打斗的描写角度与辅助设定

本节讲打斗**内部怎么描写**和**开打前后的辅助设定清单**，与「七种情境 + 节拍」互补。

### 描写角度：五大描写 + 五感

一场打斗的要素：打斗双方及亲友团、打斗环境、兵器、路人甲（嘴炮）、中立者。**想突出主角厉害，除了描主角发力，还可写对敌方的伤害、给环境带来的变化、兵器变化、路人惊叹。**

**① 五大描写角度**（动作/语言/神态/心理/外貌，分别用在双方身上）：
> 《斗破苍穹》：「俏脸铁青、眼眸冒火、咂了咂嘴」（神态）→「长腿一跃跳下巨石」（动作）→「今天绝不会轻易放过你这小混蛋」（语言）→「曼妙的娇躯…性感右腿」（外貌）→「修长双腿舞动如狂风暴雨，刮起地面枯叶」（动作 + 环境变化）。

**② 五感描写**（视觉/听觉/触觉/味觉/嗅觉，增画面感）：
> 《龙蛇演义》：「沙沙梭梭的声音，很多很杂」（听觉）→「昏黄路灯映着雪光，巷口一二十条人影」（视觉）→「打出一团白呼呼的东西」（视觉）→「鼻子闻到一股刺鼻气味」（嗅觉）→「生石灰粉！」

### 动作描写模板（可直接套）
> 反派用刀（刀叉锤盾棍棒/斧钺拳腿剑）劈主角 → 主角用刀挡 → 用脚踹对方肚子（胯下/胸口/下巴/脸）；或主角侧身（蹲下/卧倒/驴打滚）避开 + 反击。**加恰当形容词，有招式就加招式描写**（不过现在网文招式描写很少）。

### 打斗的顺序（四阶段）
1. **布局铺垫**：用环境描写（天气/景物/场地）。大场面可写万众瞩目、观战者议论（嘴炮）。《重生之都市修仙》林虎渡海复仇的咆哮 + 众人议论。
2. **快开打**：笔墨转双方，描写双方语言/神态/心理。
3. **正式开打**：重点放双方动作，不忘语言/神态/心理。
4. **打斗结束**：双方及亲友团五感描写 + 造成什么破坏。《重生之都市修仙》陈凡一拳震碎林虎全身骨骼。

### 打斗辅助设定清单（开打前自检，玄派工具）
前期铺垫（谁邀战/为何/对手厉害与可憎处/他人风凉话/主角是否发怒/策略：隐藏·秒杀·力敌·智取）→ 营造气氛（天气/景物/压抑/风雨欲来/万众瞩目）→ 主角登场（亲友/旁观者说什么、登场姿态、各方反应）→ 对手登场（同上）→ 裁判（规则/可杀人否/可钻空子）→ 众人对结果预判（亲友/高层谁看好谁不看好）→ 对手情况（态度/实力/性格背景/斩杀后遗症）→ 打斗过程（是否对话/谁先动手/胜负方式）→ 打斗时各方反应（亲友/长辈/主角/对手怎么看）→ 打斗之后（亲友/高手/对手反应 + 主角行为 + 胜利收获 + 胜利后遗症：引更强对手/被猜忌/被追杀）。

这份清单几乎是「一场打斗的全要素检查表」——卡文或觉得打斗单薄时，照着挑几项填充，立刻丰满。
