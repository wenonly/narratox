> **一句话**：女主因意外发生「怪事」（变小动物/听到心声/交换身体/时间循环），而这件怪事偏偏把她和男主绑在一起——用奇幻设定制造甜宠的天然接触契机。
>
> **写作价值 / 适用场景**：脑洞甜文短篇标准公式。适合写奇幻/系统/轻奇幻向的糖向短篇，核心是用一个「脑洞设定」代替常规的「强制接触」理由，让甜来得更新鲜。

## 一、公式骨架（导语 + 五段 + 字数）

| 阶段 | 字数 | 节拍 |
|------|------|------|
| **导语** | 50–200 字 | 背景 + 人设 + 变化 + 反转 |
| **起** | 0–2000 字 | 女主因为一个意外，身上发生怪事——变成某可爱小动物、产生幻觉幻听、听到某人心声、心声被某人听到、与某人交换身体、重复循环某个时间段等等，这件事让她很困扰 |
| **承** | 2000–5000 字 | 女主发现男主和她有同样的遭遇，或男主出现，她的怪事就会消失——从此两人有了接触，互相产生好感 |
| **冲突** | 5000–8000 字 | 女主发现男主有事瞒着自己，她不得不重新审视这段关系 |
| **转（倒叙·高潮）** | 8000–10000 字 | 从男主角度回忆：女主与男主相恋多年，分开后女主出车祸发生意外失忆，但两人再次相遇还是会爱上对方；男主为了让女主正常生活，选择消失 |
| **合（大结局）** | 10000–12000 字 | 男主消失后，女主病情痊愈、怪事消失、恢复记忆；女主找回男主，共同面对困难，重归于好 |

## 二、结构内核

```
导语(背景+反转钩子) → 起(怪事降临,困扰) → 承(怪事把两人绑定,生好感) → 冲突(男主有秘密) → 转(倒叙揭真相:失忆+男主牺牲) → 合(痊愈+找回+重圆)
```

脑洞甜文的巧思在**「怪事 = 红娘」**——把奇幻设定当成制造男女主接触的理由（变小动物被男主捡到/心声被男主听到/交换身体被迫相处），比「上下级」「房东租客」新鲜得多。而「转」段的倒叙揭示（怪事的真相是女主失忆、男主为让她正常而消失）把脑洞落回到情感内核——**奇幻是壳，深情是核**。
