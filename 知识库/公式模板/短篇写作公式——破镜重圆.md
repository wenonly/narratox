> **一句话**：分手多年后重逢，靠「酸涩的极限拉扯 + 宿命感」让两个人在误会与试探中重新走到一起——核心是「拉扯」，最忌「水到渠成」。
>
> **写作价值 / 适用场景**：破镜重圆短篇标准公式。适合写久别重逢/旧情复燃向短篇，核心卖点是「酸涩感」和「命中注定」的双重情绪。

## 一、要点

- **情感的极限拉扯**：酸涩感。
- **宿命感**：不自觉的互相吸引，注定会在一起。
- **不同时期的男女主人设变化**：性格、阅历、社会身份、经济条件、被动方与主动方，都要有前后反差。

### 男主三个时期的变化（破镜重圆的灵魂）

| 时期 | 男主状态 |
|------|----------|
| **分手时** | 无动于衷，甚至放狠话伤害女主 |
| **多年独自生活时** | 偷偷惦记女主，悔恨年轻的冲动行事 |
| **重逢后** | 纠结挣扎的内心，表面平静背后吃醋，蓄谋制造和女主相遇 |

## 二、破镜怎么写

- **误会重重，但不能有原则性问题**——误会要与人物性格自洽。
- **被迫分开**：被爱慕的第三者恶意拆散；被父母强制拆散。
- **主动分开**：性格骄傲、不屑于解释；恋爱摩擦后和平分手。

## 三、重圆怎么写

- **避免水到渠成、平淡如水的和好**，要铺垫好转折，情绪拉满。

### 导语设计
突出角色前后反差，设计反转：在一起时漠不关心 → 多年后后悔寻求。

## 四、公式骨架（起承转合，双线交织）

| 阶段 | 节拍 |
|------|------|
| **起** | 开篇重锤制造冲突：① 戏剧性（女主听到男主现任和同事吐槽自己）；② 修罗场（女主假未婚夫和男主合约女友发生矛盾） |
| **重逢拉扯** | 尴尬想逃又忍不住关注，视线频频相撞，吃醋却没有资格，明明很了解却嘴硬说不熟 |
| **承** | ① 男女主不欢而散；② 伤心回忆分手原因（男主冷漠，女主家庭巨变后分手出国，男主无动于衷） |
| **起+承** | 外力原因不得不见面（实则是男主暗中促成）：街角偶遇 / 公司共事合作 / 邻居 / 同学聚会 |
| **感情升温** | 女主嘴硬隐藏，男主满嘴嫌弃又上赶着找女主 |
| **转①** | 感情升温：女主受伤，男主第一时间为她出头，发现女主是单身（救助、帮助情节可多次出现，铺垫蓄谋已久的细节） |
| **合①** | 男主恳求女主和自己在一起，女主犹豫 |
| **转②** | 男配女配离间男女主感情（加深误会、激化矛盾、拉扯读者情绪）：一般是假未婚夫、假女友、假婚约；女配找女主放过男主、男主男配起争执打架、女配陷害女主强制她离开、男配要和女主订婚 |
| **合②** | 男女主闹矛盾，冷战 |
| **起** | 感情升温，通过配角之口揭穿男主等女主很多年、从没放弃过她（日记本 / 男主父母之口 / 社交账号 / 编入代码寻人） |
| **承** | 说清误会，反思误会，甜蜜互动（原因一般是经济、家庭条件差距大或性格原因，逻辑自洽） |
| **转③** | 男主表明心意，误会解除（避免枯燥描写：反复追问后抛出答案 / 很久后旁观者揭晓 / 主角自己发现原由） |
| **合③** | 破镜重圆，和好大结局 |
