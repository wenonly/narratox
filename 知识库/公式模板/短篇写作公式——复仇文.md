> **一句话**：用「情绪叠加」把读者的同情心堆到顶点，再让主角替天行道、逐个清算——爽感的源头是前期攒够的恨。
>
> **写作价值 / 适用场景**：复仇爽文短篇标准公式。适合写打脸/逆袭/惩恶向短篇，核心是「先虐后爽」的情绪叠加 + 层层升级的复仇行动。

## 一、要点（复仇爽文的 3 条铁律）

1. **必须先让读者产生同情心**：开篇的情绪铺垫不断叠加，且本该处理该事件的机关无能为力，所以主角必须主动出击、亲自动手。
2. **反派罪行越令人发指，主角越有理由复仇**。
3. **重点在情绪叠加和主角的复仇行动**——剧情为情绪服务。
4. **主角不能因复仇迷失自己**：他占据道德合理性，是替天行道；但他自始至终都是好人，不会拉无辜的人下水。

> 🧠 第 4 条是复仇文不招黑的底线——主角可以狠，但必须有底线（只诛首恶、不伤无辜），读者才会一直站他。

## 二、公式骨架（起承转合 + 字数）

| 阶段 | 字数 | 节拍 |
|------|------|------|
| **起** | 1000 字 | ① 开篇描写主角或亲人遭受的伤害，铺垫情绪 |
| | | ② 情绪叠加：描写反派作恶后的丑恶嘴脸 |
| | | ③ 情绪叠加：描写受害人被害后的惨状、心理阴影或精神状态 |
| | | ④ 情绪叠加：通过回忆展现受害者的无辜或善良 |
| **承** | 1000–4000 字 | ⑤ 矛盾升级，反派找茬或折磨主角，迫使主角不得不报仇，否则活不下去 |
| | | ⑥ 复仇契机：主角看到被反派伤害的人，又想到悲惨的受害者，下决心复仇 |
| | | ⑦ 复仇行动一：解决小反派 1 |
| | | ⑧ 复仇行动一：解决小反派 2 |
| | | ⑨ 复仇行动一：解决小反派 3 |
| | | ⑩ 调查 boss 信息，准备解决 boss |
| **转** | 4000–8000 字 | ⑪ 复仇行动二：吸引反派注意 |
| | | ⑫ 复仇行动二：抛出鱼饵让反派入局 |
| | | ⑬ 复仇行动二：激怒反派 |
| | | ⑭ 复仇行动三：反派上当 |
| | | ⑮ 复仇行动三：主角复仇，反派痛苦难过 |
| **合** | 8000–10000 字 | ⑯ 揭露反派的所作所为，反派受到惩罚；主角和受害者过上新生活 |

## 三、结构内核

```
起(情绪叠加 4 连，攒恨) → 承(复仇行动一：清小怪，热身) → 转(复仇行动二三：做局+收网 boss) → 合(公开处刑+新生)
```

> 🧠 复仇文是**「台阶式爽感」**：起段把恨攒满（4 次叠加），承段先清小反派（小额爽点热身），转段做局收 boss（大额爽点高潮），合段公开处刑（社会性死亡，爽感最大化）。每一级都比上一级爽。

---

### 🧠 整理思考

**结构判断**：公式型。16 个节拍收进起承转合表。

**知识提炼**：
1. **爽感 = 前期攒的恨 × 后期打的爽**。恨攒不够，打再狠也不爽。
2. **台阶式升级**：小反派 → boss → 公开处刑，每一级加码。
3. **主角必须有道德底线**（替天行道、不伤无辜），否则招黑。

**使用建议**：起段舍得花笔墨「攒恨」（4 次叠加一次不能少）；复仇行动按「清小怪→做局→收网→公开处刑」四级升级。换皮：校园/职场/宅斗/玄幻都能套。

**关联**：与 [[短篇写作公式——世情虐渣打脸文]] 是近亲（都先虐后爽、打脸复仇）。情绪叠加手法与 [[短篇写作公式——虐文]] 的「越努力越失败」相反——这里是「越压迫越该反击」。
