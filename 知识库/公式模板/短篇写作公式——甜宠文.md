> **一句话**：靠男女主反差人设 + 反复肢体接触 + 细节糖，在 1 万字内走完「起承转合」的情绪过山车，全程高甜。
>
> **写作价值 / 适用场景**：甜宠短篇的标准公式。适合写无虐/轻虐的糖向短篇，核心是把「甜」做成可复制的节拍——每一段都有明确的情绪方向（上行/下行）。

## 一、人设要点

- **男女主必须有反差**：男主高冷 → 女主就要是甜妹（反之亦然）。
- **用一个牵扯把两人绑在一起**：上下级、医生与病人、房东与租客。
- **制造多次肢体接触**：递文件、开车门等不经意的触碰。
- **用小细节塑造人物**：男主记得女主不吃葱、奶茶要去冰、中午要补觉。

## 二、要点（甜宠的 4 个秘诀）

1. **误会后的「和好」必须有阻碍**：极端天气、天灾人祸、旧病复发——阻碍越狠，和好越甜。
2. **男主台词切忌油腻**。
3. **开头男主主动，结尾女主反撩回去**，更有看点。
4. 全程靠「情绪上行/下行」驱动，而不是复杂剧情。

## 三、公式骨架（起承转合 + 字数 + 情绪）

| 阶段 | 字数 | 节拍 | 情绪 |
|------|------|------|------|
| **起** | 1000 字 | ① 开篇制造意外相遇，铺垫情感（女主赶地铁撞到男主，发现他是自己顶头上司） | ↑ |
| **承** | 1000–4000 字 | ② 被迫见面，互动升温（合作项目每天开会，男主表面冷冰冰，其实会安慰女主、带早餐、聚会挡酒） | ↑ |
| | | ③ 产生小误会（女主刚有好感，却看到男主和女配关系密切、卿卿我我） | ↓ |
| | | ④ 女配故意装亲密，女主想约男主问清楚，男主却因某事错过约会，嫌隙变大 | ↓ |
| | | ⑤ 女主心灰意冷打算结束合作，男主得知后撕碎文件 | ↓ |
| **转** | 4000–8000 字 | ⑥ 男主冒雨去找女主解释，结果出了车祸 | ↓ |
| | | ⑦ 女主来医院看望昏迷男主，发现手机屏保是自己 | ↑ |
| | | ⑧ 男主醒来，两人对峙，女主发现真相 | ↑ |
| | | ⑨ 男主表白，警告女配，主动与女配划清界限 | ↑ |
| **合** | 8000–10000 字 | ⑩ 女主发现男主备忘录里记着自己所有喜好；男主拿出备好的彩礼、婚房、婚车（都写了女主名字），单膝跪地求婚，女主答应 | ↑ |

## 四、情绪曲线（公式核心）

```
起(↑) → 承(↑↓↓，先甜后虐) → 转(↓↑↑↑，触底反弹) → 合(↑，大甜收尾)
```

> 🧠 关键在「转」段的**触底反弹**：先把情绪砸到最低（车祸/误会），再用屏保、真相、表白连续三连拉升。没有低谷，高峰就不甜。

---

### 🧠 整理思考

**结构判断**：公式型。按「人设 → 要点 → 起承转合表 → 情绪曲线」重排，把 10 个节拍连同字数、情绪方向收进一张表——这是甜宠公式最该被一眼看懂的部分。

**知识提炼**：
1. **甜宠 = 反差人设 + 细节糖 + 情绪过山车**，三者缺一不可。
2. **「转」段必须触底反弹**——低谷越深，反弹越甜。
3. **细节糖是甜宠的灵魂**（不吃葱/奶茶去冰/备忘录），比大段告白更戳人。

**使用建议**：直接照着起承转合表填自己的设定，每段卡住字数和情绪方向即可成篇。换皮不换骨：把「上下级」换成任何强制接触关系都成立。
